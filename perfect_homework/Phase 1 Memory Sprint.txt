Element 3 · To'g'ri / Noto'g'ri · Nima uchun a ≠ 0 muhim

Savol: Tenglama 0·x² + 5x – 2 = 0 kvadrat tenglama hisoblanadi.
The equation 0·x² + 5x – 2 = 0 is a quadratic equation.

Variantlar:

- To'g'ri
- Noto'g'ri ✓

Tushuntirish (javobdan keyin): Agar a = 0 bo'lsa, x² yo'qoladi va tenglama chiziqli (linear) bo'lib qoladi. Kvadrat tenglamada a ≠ 0 shart.
If a = 0, the x² disappears and the equation becomes linear. A quadratic requires a ≠ 0.

Teglar: [Bloom: L2 | PISA: L2]
Ko'nikma: tanqidiy fikrlash, xotira

Element 5 · Ha / Yo'q / Aniq emas · Ildizlar haqida umumiy da'vo

Savol: Quyidagi fikr to'g'rimi?
Evaluate the statement:

▎ "Har qanday kvadrat tenglama aniq ikkita haqiqiy ildizga ega."
▎ "Every quadratic equation has exactly two real roots."

Variantlar:

- Ha (Yes)
- Yo'q (No) ✓
- Aniq emas (Not given)

Tushuntirish (javobdan keyin): Kvadrat tenglama 0, 1, yoki 2 ta haqiqiy ildizga ega bo'lishi mumkin — x² = d misolida d < 0 bo'lsa, haqiqiy ildiz yo'q.
A quadratic can have 0, 1, or 2 real roots — for x² = d with d < 0, there are no real roots.

Teglar: [Bloom: L2 | PISA: L2]
Ko'nikma: tanqidiy fikrlash, xotira
