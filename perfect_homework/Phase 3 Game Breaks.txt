O'yin 1 — Adaptiv Viktorina (8 ta element zaxirasi, har bir seansda 5-7 ta ko'rsatiladi)

Har bir javobda suratga olish talab etiladi. IRT algoritmi avvalgi to'g'rilikka qarab keyingi elementni tanlaydi. Oson darajadan boshlab, u yerdan oshadi/kamaytiradi.

Oson daraja — L2

Element A1

- Savol: Yeching: x² = 81
- Kutilgan: x₁,₂ = ±9
- Suratda ko'rsatilishi kerak: x = ±√81 = ±9
- Teglar: [Bloom: L2 | PISA: L1]

Element A2

- Savol: Yeching: x² = 0
- Kutilgan: x = 0 (bitta ildiz)
- Suratda ko'rsatilishi kerak: x² = 0 → x = 0
- Teglar: [Bloom: L2 | PISA: L1]

Element A3

- Savol: Yeching: x² + 16 = 0
- Kutilgan: haqiqiy ildiz yo'q (no real roots)
- Suratda ko'rsatilishi kerak: x² = –16 < 0 → haqiqiy ildiz yo'q
- Teglar: [Bloom: L2 | PISA: L2]

O'rta daraja — L3

Element A4

- Savol: Yeching: x² = 20
- Kutilgan: x₁,₂ = ±2√5
- Suratda ko'rsatilishi kerak: x = ±√20 = ±√(4·5) = ±2√5
- Teglar: [Bloom: L3 | PISA: L2]
- Ko'nikma: irrational ildizlarni soddalashtirish

Element A5

- Savol: Yeching: x² – 49 = 0
- Kutilgan: x₁ = 7, x₂ = –7
- Suratda ko'rsatilishi kerak: x² – 49 = (x – 7)(x + 7) = 0 → x = ±7
- Teglar: [Bloom: L3 | PISA: L2]
- Ko'nikma: kvadratlar ayirmasi

Element A6

- Savol: Yeching: x² + 5x + 6 = 0
- Kutilgan: x₁ = –2, x₂ = –3
- Suratda ko'rsatilishi kerak: p + q = 5, p · q = 6 → (2, 3); (x + 2)(x + 3) = 0 → x = –2 yoki –3
- Teglar: [Bloom: L3 | PISA: L3]
- Ko'nikma: ko'paytuvchi-juft usuli (Oldindan ko'rish 2-Paneli ① da o'rgatilgan)

Qiyin daraja — L3-L4

Element A7

- Savol: Yeching: x² + 10x – 24 = 0
- Kutilgan: x₁ = 2, x₂ = –12
- Suratda ko'rsatilishi kerak: p + q = 10, p · q = –24 → (–2, 12); (x – 2)(x + 12) = 0 → x = 2 yoki –12
- Teglar: [Bloom: L3 | PISA: L3]
- Eslatma: bu darslikning o'z 1-muammosi. Oldindan ko'rishdan to'liq aylanib keladi.

Element A8

- Savol: Yeching: x² – 3x – 10 = 0
- Kutilgan: x₁ = 5, x₂ = –2
- Suratda ko'rsatilishi kerak: p + q = –3, p · q = –10 → (–5, 2); (x – 5)(x + 2) = 0 → x = 5 yoki –2
- Teglar: [Bloom: L4 | PISA: L3]
- Ko'nikma: aralash ishorali ko'paytuvchi-juft usuli

Suratga olish mezonlari (har bir elementga qo'llaniladi)

AI vision baholovchi suratni quyidagilar uchun tekshiradi:

1. O'quvchi asl tenglamani yozgan
2. Yechish usulini ko'rsatuvchi kamida bitta oraliq qadam
3. Yakuniy javob kutilganiga mos keladi
4. Yozuv o'qilishi mumkin (aks holda → qayta suratga olish so'rovi)

4 tadan 3 ta bo'lsa o'tadi. Javob to'g'ri bo'lsa, lekin ish ko'rsatilmagan bo'lsa, qisman ball (o'qituvchi panelida "ishni ko'rsating" belgisi bilan belgilansin).
