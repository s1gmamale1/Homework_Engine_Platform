# SKILLS — Homework Production Process

---

## Assignment Handling

**If given a specific chapter:** Produce one docx for that chapter.
**If given a subject with no chapter:** Produce docx for EVERY chapter in the textbook, sequentially. One file per chapter.
**If unclear:** Ask: "How many homeworks should I produce? Specific chapters or the whole book?"

---

## Pre-Flight (Once Per Subject, Not Per Chapter)

You already know the framework from SOUL.md. Before your first chapter:

1. **Identify the Subject Family** from the family table in SOUL.md
2. **Read the full Library Framework** at the path in Repo Map — refine your knowledge of this specific family's mechanic emphasis, boss format, pedagogical principles, grade-band overrides
3. **Read the Game Catalog** — understand the specific content format each game expects
4. **Scan the textbook** — get the TOC, understand chapter structure, count chapters if doing whole book

After pre-flight, you should know which 3 games to use in Phase 3, the boss format, and the chapter list.

**Do NOT repeat pre-flight for every chapter.** Only re-read framework docs if you compacted context and lost the information.

---

## Production — One Chapter → One Docx

### Output file: `{Subject}_Grade{N}_Ch{N}_{lang}.docx`

For each chapter, produce a docx with this structure:

---

### PRE-SESSION 0-A — MAVZU KO'RIGI (Theme Preview) — 2-3 daqiqa

Student-paced, no scoring, no timer. Visual-first, first-person POV ("you" not "the student").

**8 required components:**
1. **Darslik mazmuni:** Cleaner reframing of what the chapter covers. Not a rewrite — a refocusing.
2. **Yaxshiroq tushuntirish:** Concepts the textbook assumes, clarified. Fill comprehension gaps.
3. **Misollar:** Additional worked examples beyond the textbook, step-by-step.
4. **Hayotiy tadqiqot:** Origin story of the concept. Who discovered it? What problem did it solve?
5. **Shaxsiy bog'lanish:** First-person hook. "Esingda bormi...?" / "Balki sen ham hayron bo'lgansan..."
6. **Nima uchun muhim:** Real-world relevance — not "it's on the exam" but genuine why-you-need-this.
7. **Sanoat qo'llanishi:** How this concept is used in real industries, jobs, technology.
8. **Qo'shimcha materiallar:** Links/references to videos, articles, simulators (any language OK).

End with transition: *"Boshlashdan oldin — sizga kerak bo'ladigan asosiy g'oyalar."*

---

### PRE-SESSION 0-B — FLESH-KARTALAR (Flash Cards) — 1-2 daqiqa

Reference deck — no scoring, no quiz. One concept per card.

Must contain:
- **All key formulas** the student will need in this homework
- **All key concepts** (definitions, terminology, classifications)
- **All key rules** (conditions, exceptions, boundaries)

End with: **[ MENING UY VAZIFAMNI BOSHLASH ]** button marker.

---

### PHASE 1 — XOTIRA SPRINT (Memory Sprint) — 2 daqiqa

- **5 rapid-fire recall questions from PRIOR chapters** (not current topic)
- If Chapter 1 (no prior chapters), use prerequisite knowledge from the previous grade
- 10-15 seconds per question
- Bloom's L1 (Remember). PISA L1.
- Mix: terminology, formula recall, definitions
- **Include answer key after all questions**
- XP note: 100 XP per correct + streak bonus

---

### PHASE 2 — HIKOYA REJIMI (Story Mode) — 5-7 daqiqa

- **3 segments** delivering the chapter's core concepts
- **CPA progression:** Segment A = Concrete (real objects, familiar scenarios), Segment B = Pictorial (diagrams, models, number lines), Segment C = Abstract (formulas, rules, symbolic notation)
- Each segment: 1-3 paragraphs of narrative with Uzbek cultural context
- **Each segment ends with a Checkpoint** (1-2 comprehension questions + answers)
- Bloom's L1-L2 (Remember/Understand). PISA L1-L2.
- Must pass "Stranger Test" — a newcomer can follow the narrative
- XP: 0 (pure exploration, no scoring)

---

### PHASE 3 — O'YIN TANAFUSLARI (Game Breaks) — 6-9 daqiqa

- **3 games, 2-3 minutes each**
- **Rule: ≥1 from Interactive Catalog, ≥2 from Default Pool**
- Select from this family's mechanic emphasis table (primary first, secondary if fits)
- 4-6 items per game
- Bloom's L1-L3. PISA L1-L2.
- **Include answer key for each game**

Format each game according to its type:
- **Tile Match:** Left column → Right column pairs + distractors
- **Why Chain:** Seed question → 2-3 "why" follow-ups at increasing depth
- **Adaptive Quiz:** 5-7 questions at varied difficulty
- **Sentence Fill:** Text with ___ blanks + word bank
- **Speed Match:** Rapid Q&A pairs (10 sec each)
- **Memory Palace:** 5 concepts placed at 5 Registan Square locations
- **Error Analysis:** Deliberately wrong worked example → student finds the error
- **Mystery Box:** Hidden problem type identification + solve
- **Notebook Capture:** Prompt for handwritten work + rubric

XP: 50-400 per game depending on mechanic.

---

### PHASE 4 — HAQIQIY HAYOT (Real-Life Challenge) — 3-5 daqiqa

- **1 scenario** using Uzbek cultural context (bazaar, farm, school, family)
- Student takes a role (inspector, consultant, engineer, helper)
- **3 open-ended questions:** 1 application, 1 analysis, 1 transfer
- Bloom's L3-L4. PISA L2-L3.
- **Include answer key with explanations**
- XP: 100-400 rubric-based

---

### PHASE 5 — MUSTAHKAMLASH (Consolidation / Mnemonic Lock) — 2-3 daqiqa

- Lock key concepts before Final Boss
- Use ONE Buzan technique appropriate to the content:
  - **Memory Palace** (default): 5 locations at Registan Square, each tied to a key concept
  - **Peg System**: Number-Shape for math rules
  - **Mnemonic/Acronym**: SMASHIN' SCOPE quality gate
  - **Flashcard list**: 5-6 concept → definition pairs
- Apply SMASHIN' SCOPE: make it vivid, humorous, exaggerated, colorful
- XP: 0 (preparation phase)

---

### PHASE 6 — YAKUNIY BOSS (Final Boss) — 5-10 daqiqa

- Follow the **family's specific boss format** from the Library Framework
- **HP system:** G5-8 = 100 HP total
- **Question distribution:** 40% Easy (-10 HP), 40% Medium (-20 HP), 20% Hard (-30 HP)
- **Bloom's progression:** Apply → Analyze → Evaluate
- **PISA progression:** L2 → L3 → L4
- **Format:** Open-ended (no MC for G6+; G5 allows up to 30% MC)
- **Each question includes:**
  - Question text
  - Socratic hints (2-3 per question) — hint cost: -5 to -15 HP depending on grade
  - Rubric with HP damage values (full/partial/wrong)
  - Expected answer
- **Mastery stars:** 1★ = defeated | 2★ = ≤2 attempts, >50% HP | 3★ = first attempt, no hints, >80% HP

---

### PHASE 7 — AKS ETTIRISH (Reflection) — 2-3 daqiqa

- **Session summary:** What was covered, key takeaways
- **3 reflection prompts:** What was hardest? Which method helped most? What would you do differently?
- **Spaced repetition note:** When to review (strong concepts: 7 days, moderate: 3 days, weak: next day)
- **Next lesson preview:** Title of next chapter
- XP: 0 (metacognitive, not scored)

---

## Context Management

**If context reaches ~90% capacity:**
1. Write current progress to STATUS.md (which chapters are done, which is next)
2. Save any in-progress work
3. Summarize/compact your context
4. Resume from STATUS.md — re-read SOUL.md framework summaries (they're compact enough to reload)
5. Continue from the next unfinished chapter

**Do NOT restart from scratch.** STATUS.md is your resume point.
