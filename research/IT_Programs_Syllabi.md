# Research Brief: Phase 3 — Uzbekistan IT Program Syllabi (1M Coders & IT Park)

This report details the syllabi for the "One Million Uzbek Coders" program (in partnership with Udacity) and the resident courses offered by IT Park Academy/University.

---

## 1. One Million Uzbek Coders (OMUC) - Udacity Tracks

The OMUC initiative provides 4 primary tracks, each divided into an **Introductory Stage** (120 hours) and an **Advanced Nanodegree Stage** (for grant winners).

### A. Data Analysis Track
Focuses on the data science lifecycle, teaching students how to uncover insights and communicate findings.
*   **Languages & Tools:** Python, SQL, NumPy, Pandas, Matplotlib, Jupyter Notebooks.
*   **Module-by-Module Syllabus:**
    1.  **The Data Analysis Process:** Introduction to questioning, wrangling, exploring, and analyzing data.
    2.  **SQL for Data Analysis:** Basic commands, Joins, Aggregations, Subqueries, and Cleaning.
    3.  **Python Libraries for Data Analysis:** Using NumPy and Pandas for 1D and 2D data manipulation.
    4.  **Practical Statistics:** Descriptive statistics, probability, and hypothesis testing.
    5.  **Data Visualization:** Creating impactful charts and stories with Matplotlib and Seaborn.
*   **Key Project:** Investigating a real-world dataset (e.g., TMDb movie data) to share findings.

### B. Android Development Track
Developed in collaboration with Google, focusing on modern Android app architecture.
*   **Languages & Tools:** Kotlin, Android Studio, Android Jetpack (Navigation, Room, ViewModel), Retrofit, Glide.
*   **Module-by-Module Syllabus:**
    1.  **App Basics:** Setting up Android Studio, building the first app (e.g., Dice Roller).
    2.  **UI Design:** Mastering Layouts (ConstraintLayout), Data Binding, and ViewModels.
    3.  **Navigation:** Implementing multi-screen navigation using the Navigation library and Fragments.
    4.  **App Lifecycle:** Managing Activity and Fragment states during rotation and backgrounding.
    5.  **Persistence & Networking:** Using Room for local databases and Retrofit for fetching API data.
*   **Key Project:** Building a Trivia app or a complex Task Manager with local storage.

### C. Front-End Development Track
Focuses on client-side web development and responsive user interfaces.
*   **Languages & Tools:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 4, jQuery, Flexbox, CSS Grid.
*   **Module-by-Module Syllabus:**
    1.  **Web Foundations:** Semantic HTML, CSS Box Model, and responsive design with Flexbox/Grid.
    2.  **CSS Frameworks:** Rapid prototyping with Bootstrap 4.
    3.  **JavaScript Basics:** Data types, loops, conditionals, functions, and scope.
    4.  **DOM Manipulation:** Using vanilla JS and jQuery to dynamically update content.
    5.  **Advanced (Nanodegree level):** Webpack, Sass, Asynchronous JavaScript (fetch API), and **React.js**.
*   **Key Project:** Creating a professional business landing page and a Weather Journal App using APIs.

### D. Full-Stack Development Track
Focuses on backend development, database management, and API security.
*   **Languages & Tools:** Python 3, Flask, PostgreSQL, SQLAlchemy (ORM), Docker, AWS, Postman.
*   **Module-by-Module Syllabus:**
    1.  **SQL and Data Modeling:** Linking Python to PostgreSQL using SQLAlchemy; CRUD operations.
    2.  **API Development:** Building RESTful APIs with Flask; handling HTTP methods and status codes.
    3.  **Identity and Access Management:** Implementing JWT (JSON Web Tokens) and Auth0 for security (RBAC).
    4.  **Server Deployment:** Containerization with Docker and deployment to AWS (Elastic Beanstalk).
*   **Key Project:** Developing a "Casting Agency" API with multiple user roles and permissions.

## 2. IT Park Academy & IT Park University (ITPU)

IT Park offers vocational courses through IT Academy and degree programs through ITPU.

### A. Foundation Program (Preparatory)
*   **Focus:** English Proficiency (IELTS 4.5+ prep) and Mathematics.
*   **IT Literacy:** Microsoft Office (Word, Excel), OS management, and Digital Safety.
*   **Duration:** 3–6 months.

### B. Game Development Course
*   **Academy:** GameDevHQ Academy (launched Dec 2024) and Xsolla Academy.
*   **Tools:** Unity, Unreal Engine 5, Blender.
*   **Syllabus:**
    *   **C# for Unity:** Logic, Physics, and Scripting.
    *   **C++ for Unreal:** Professional game architecture.
    *   **Game Design:** Level design, mechanics, and UI/UX for games.
    *   **3D Art:** Modeling, texturing, and animation integration.

### C. Cybersecurity Program
*   **Level:** Bachelor's (ITPU) and Specialist Certifications.
*   **Languages & Tools:** Python, Java, Linux (Kali/Ubuntu), Wireshark, Metasploit.
*   **Syllabus:**
    *   **Year 1-2:** Intro to Digital Tech, Python, Software Testing, and Network Basics.
    *   **Specializations:**
        *   **Security Operations (SOC):** Monitoring and responding to threats.
        *   **Network Security:** Firewalls, VPNs, and Intrusion Detection.
        *   **DevSecOps:** Integrating security into the CI/CD pipeline.
        *   **Risk Management:** Governance and compliance.
