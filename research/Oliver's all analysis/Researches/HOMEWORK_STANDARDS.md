## 1. CORE PHILOSOPHY & NON-NEGOTIABLES

### 1.1 The NETS Learning Pyramid

Every homework session MUST map to Bloom's Taxonomy through the NETS Learning Pyramid:

```
                    ┌───────────┐
                    │  TRANSFER │ ← Real-Life Challenges (PISA Level 5-6)
                   ┌┴───────────┴┐
                  │   ANALYZE    │ ← Why Chain, Mystery Box (PISA Level 4)
                 ┌┴──────────────┴┐
                │     APPLY       │ ← Adaptive Quiz, Final Boss (PISA Level 3)
               ┌┴─────────────────┴┐
              │    UNDERSTAND     │ ← Story Mode, Tile Match (PISA Level 2)
             ┌┴───────────────────┴┐
            │      REMEMBER       │ ← Memory Sprint, Flashcards (PISA Level 1)
           ┌┴─────────────────────┴┐
           │    ENGAGE (Foundation) │ ← Gamification, Flow State
           └───────────────────────┘
```

**NON-NEGOTIABLE:** Every homework session MUST include activities spanning at least 4 levels of the pyramid.

---

### 1.2 The 3 Sacred Principles

#### Principle 1: Textbook is Source of Truth, NETS is the Transformer

| What Textbook Does | What NETS Does |
|-------------------|----------------|
| Provides core concepts | Forces REASONING with concepts |
| Presents information | Creates active engagement |
| Shows solved examples | Requires student to CONSTRUCT solutions |

**STANDARD:** Homework MUST NOT change core concepts from textbooks. It MUST adapt delivery method to active, gamified, AI-personalized learning.

**Example (Mathematics - Fractions):**
```
Textbook: "½ + ½ = 1. Here's the rule..."
NETS: "Alisher and Malika share a pizza. The pizza has 8 slices. 
       If Alisher eats 4 slices and Malika eats 4 slices, 
       what fraction did each eat? Drag the slices to show your answer."
```

---

#### Principle 2: Flow State is Mandatory

Based on Csikszentmihalyi (1990), every student MUST be maintained in the flow channel:

```
Challenge Level
    ↑
    │         ANXIETY (Overwhelmed - Student Quits)
    │              ╱
    │            ╱
    │          ╱  ╔═══════════╗
    │        ╱    ║ FLOW      ║ ← TARGET ZONE
    │      ╱      ║ 70-80%    ║
    │    ╱        ║ Success   ║
    │  ╱          ╚═══════════╝
    │╱___________________________
    │    SKILL Level    ╲
    │                     ╲
    │                       BOREDOM (Underwhelmed - Student Quits)
```

**STANDARD:** AI MUST adjust difficulty every 3-5 questions:
- Below 60% success rate → Reduce difficulty by 1 tier
- Above 90% success rate → Increase difficulty by 1 tier
- 70-80% success rate → Maintain current tier

---

#### Principle 3: Intrinsic Motivation > Extrinsic Rewards

All rewards MUST celebrate LEARNING PROGRESS, not compliance.

| ✅ Intrinsic (DO) | ❌ Extrinsic (DON'T) |
|------------------|---------------------|
| "You mastered fractions!" | "You logged in 7 days!" |
| "Your reasoning improved!" | "You clicked 100 times!" |
| "You helped a classmate!" | "You watched an ad!" |

**STANDARD:** Zero pay-to-win mechanics. Zero real-money purchases. Every reward is earned through demonstrated learning.

### 2.2 Step 1: Memory Sprint (2 minutes)

**Purpose:** Activate prior knowledge, prime neural pathways, connect old → new

**Research Basis:** Retrieval Practice (Roediger & Karpicke, 2006) - active recall strengthens memory traces 300% more than re-reading.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Duration** | Exactly 60 seconds (visible countdown) |
| **Question Count** | 5-8 questions |
| **Content Source** | PREVIOUS chapters ONLY (not current lesson) |
| **Question Type** | Pure recall - NO multiple choice for Grades 3+ |
| **Hints** | NONE (this is recall, not learning) |
| **XP Award** | 100 XP per correct + 10 XP streak bonus |

**UI Requirements:**
- Large visible timer (00:60 countdown)
- Current streak display (🔥🔥🔥 = 3)
- Immediate feedback (green flash/red shake)
- XP animation on each correct answer

**Adaptation Rules:**
```
IF accuracy >= 80% → Student ready for new content
IF accuracy 60-79% → Proceed but flag for teacher review
IF accuracy < 60% → System routes to remediation BEFORE new content
```

**Subject Examples:**

**Mathematics (Grade 5 - Fractions):**
```
Q: What is the formula for calculating the area of a triangle?
A: A = ½ × base × height

Q: Convert ¾ to a decimal.
A: 0.75
```

**Reading (Grade 6 - Story Elements):**
```
Q: What is the main character's name from yesterday's story?
A: Alisher

Q: What problem did Alisher face in the story?
A: He couldn't solve the fraction puzzle to open the treasure chest.
```

**Science (Grade 7 - Photosynthesis):**
```
Q: What gas do plants absorb from the air?
A: Carbon dioxide (CO2)

Q: What do plants produce during photosynthesis?
A: Glucose (sugar) and oxygen
```

### 2.3 Step 2: Story Mode (5-7 minutes)

**Purpose:** Deliver new content through narrative, leverage brain's natural story processing

**Research Basis:** Narrative retention is 22x higher than abstract presentation (Willingham, 2004). Stories activate multiple brain regions simultaneously.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Duration** | 5-7 minutes total |
| **Segments** | 3 segments × 90 seconds each |
| **Comprehension Checks** | 1-2 questions between each segment |
| **Media Format** | AI-generated video OR illustrated panels |
| **Cultural Integration** | MUST feature Uzbek locations, names, scenarios |

**Structure:**
```
┌─────────────────────────────────────────┐
│  📖 STORY MODE: Chapter 4.2             │
│  "The Young Explorer of Samarkand"      │
├─────────────────────────────────────────┤
│                                         │
│  SEGMENT 1 (90 sec)                     │
│  → Story introduction + concept preview │
│  → Comprehension Check (30 sec)         │
│                                         │
│  SEGMENT 2 (90 sec)                     │
│  → Concept development with examples    │
│  → Comprehension Check (30 sec)         │
│                                         │
│  SEGMENT 3 (90 sec)                     │
│  → Real-world connection                │
│  → Transition to Game Breaks            │
│                                         │
└─────────────────────────────────────────┘
```

**Grade-Level Narrative Framework:**

| Grade | Semester Narrative | Cultural Elements |
|-------|-------------------|-------------------|
| 3 | "The Young Explorer of Samarkand" | Registan, Silk Road, traditional crafts |
| 4 | "The Silk Road Expedition" | Caravans, trade, cultural exchange |
| 5 | "The Cotton Harvest Challenge" | Agriculture, family work, rural life |
| 6 | "Building the Registan" | Architecture, mathematics in design |
| 7 | "The Amir Temur Campaign" | History, strategy, leadership |
| 8 | "Modern Tashkent" | Urban life, technology, progress |
| 9-11 | Subject-specific narratives | Professional applications |

**Content Adaptation Rules:**

1. **Preserve Core Concepts:** Story MUST NOT change textbook concepts
2. **Add Narrative Layer:** Wrap concepts in character-driven story
3. **Embed Questions:** Questions arise naturally from story context
4. **Cultural Authenticity:** Use real Uzbek locations, historical references

**Example (Mathematics - Grade 5, Fractions):**

```
Textbook Concept: "Adding fractions with like denominators"

Story Adaptation:
"Alisher and his grandmother are preparing plov for a family gathering.
The recipe calls for ¾ cup of rice for 4 people. But 8 people are coming!

'How much rice do we need now?' his grandmother asks.

Alisher thinks: 'If 4 people need ¾ cup, then 8 people need...
¾ + ¾ = 6/4 = 1½ cups!'

'Correct!' his grandmother smiles. 'Now show me how you calculated it.'"
```

#### Game 3A: Tile Match

**Research:** Dual Coding Theory (Paivio, 1986) - visual + verbal encoding produces 40% better retention.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Grid Size** | 8-12 tiles (4-6 pairs) |
| **Timer** | 2-3 minutes (visible countdown) |
| **XP Award** | Perfect: 300 XP | Partial: 100-250 XP |
| **Feedback** | Correct: tiles disappear + chime | Incorrect: tiles shake -5 seconds |

**Mechanics:**
1. Student clicks two tiles to attempt match
2. Correct match → tiles disappear, +100 XP, visual/audio feedback
3. Incorrect match → tiles shake, -5 seconds from timer
4. Complete all pairs before time runs out

**Expected Improvement:** 25-40% faster concept recognition

**Subject Examples:**

**Mathematics (Fractions):**
```
Tile A: "½"          Tile B: [Visual: 1 of 2 parts shaded]
Tile C: "⅓"          Tile D: [Visual: 1 of 3 parts shaded]
Tile E: "¼"          Tile F: [Visual: 1 of 4 parts shaded]
```

**Science (Photosynthesis):**
```
Tile A: "Chloroplast"    Tile B: [Diagram: plant cell with chloroplast highlighted]
Tile C: "Stomata"        Tile D: [Diagram: leaf underside with stomata]
Tile E: "Glucose"        Tile F: [Chemical formula: C₆H₁₂O₆]
```

**Reading (Literary Devices):**
```
Tile A: "Metaphor"       Tile B: "The classroom was a zoo"
Tile C: "Simile"         Tile D: "Her smile was like sunshine"
Tile E: "Personification"| Tile F: "The wind whispered secrets"
```

#### Game 3B: Sentence Fill (Cloze Test)

**Research:** Cloze Testing (Taylor, 1953) - reliable measure of comprehension, 35% vocabulary acquisition increase.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Passage Length** | 3-5 sentences |
| **Blank Count** | 3-5 blanks per passage |
| **Timer** | 2-3 minutes |
| **Word Bank** | Grades 2-4: Provided | Grades 5-7: Partial | Grades 8-11: None |
| **XP Award** | 100 XP per correct fill |

**Difficulty Progression:**

| Grade | Word Bank | Blank Frequency | Sentence Complexity |
|-------|-----------|-----------------|---------------------|
| 2-4 | Full (6 options, 3 blanks) | Every 3rd word | Simple sentences |
| 5-7 | Partial (5 options, 4 blanks) | Every 4th word | Compound sentences |
| 8-11 | None (free recall) | Every 5th word | Complex sentences |

**Subject Examples:**

**Reading (Grade 5 - Silk Road):**
```
Merchants traveled along the [_______].
The journey took many [_______]
because the distance was so [_______].

Word Bank: spices | weeks | vast | gold | days | steep

Answers: Silk Road | weeks | vast
```

**Science (Grade 7 - Photosynthesis):**
```
Plants absorb [_______] from the air through tiny openings
called [_______]. Using sunlight energy, they convert it
into [_______] and release [_______] as a byproduct.

Word Bank: (Grades 5-7: Partial bank provided)
           (Grades 8-11: No bank - free recall)

Answers: CO₂/carbon dioxide | stomata | glucose/sugar | oxygen
```

#### Game 3C: Why Chain (Socratic Dialogue)

**Research:** Elaborative Interrogation (Pressley et al., 1992) - asking "why" questions produces 2 PISA sub-level increase in reasoning ability.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Chain Length** | 3-5 "why" levels minimum |
| **Timer** | 3 minutes |
| **AI Response** | Must acknowledge + probe deeper each level |
| **XP Award** | 150 XP per level completed |
| **Hint System** | Available with -10 XP penalty |

**AI Response Pattern:**
```
Level 1: "Why do plants need sunlight?"
Student: "To grow."
AI: "✓ Good start! But WHY does sunlight help them grow?

Level 2: What does sunlight PROVIDE to the plant?
Student: "Energy."
AI: "✓✓ Excellent! What do plants DO with that energy?

Level 3: How do plants USE the energy from sunlight?
Student: "They make food from CO2 and water."
AI: "✓✓✓ Perfect! You've described photosynthesis!"
```

**Expected Improvement:** 2 PISA sub-level increase in science reasoning

**Subject Examples:**

**Science (Grade 7 - Photosynthesis):**
```
L1: Why do plants need sunlight?
L2: What does sunlight provide?
L3: What do plants make with that energy?
L4: Why is this process called "photosynthesis"?
L5: What would happen if there was no sunlight for a week?
```

**History (Grade 8 - Silk Road):**
```
L1: Why did the Silk Road develop through Samarkand?
L2: What geographic features made this route favorable?
L3: Why was Samarkand more than just a trading stop?
L4: How did the Silk Road change the cultures it connected?
L5: Why did the Silk Road eventually decline?
```

**Mathematics (Grade 6 - Fractions):**
```
L1: Why do we need common denominators to add fractions?
L2: What does the denominator represent?
L3: Why can't we just add ½ + ⅓ = ⅖?
L4: How would you explain this to a younger student?
```

#### Game 3D: Mystery Box (Interleaving)

**Research:** Interleaving (Rohrer & Taylor, 2007) - mixing problem types produces 43% better transfer to novel situations.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Box Count** | 3-5 mystery boxes |
| **Content** | Mixed problem types from current + previous chapters |
| **Timer** | 2-3 minutes |
| **XP Award** | 200 XP per correct identification |

**Mechanics:**
1. Student sees 3-5 closed boxes
2. Click a box → reveals a problem
3. Student must FIRST identify what type of problem it is
4. THEN solve it
5. Immediate feedback on both identification and solution

**Subject Examples:**

**Mathematics (Grade 5 - Mixed Operations):**
```
Box 1: "15 + 27 = ?" → Student identifies: "Addition"
Box 2: "A pizza has 8 slices. You eat 3. What fraction remains?" → Student identifies: "Fraction subtraction"
Box 3: "What is ¾ as a decimal?" → Student identifies: "Fraction-decimal conversion"
```

**Reading (Grade 6 - Literary Analysis):**
```
Box 1: "The wind whispered through the trees." → Student identifies: "Personification"
Box 2: "Her smile was like sunshine." → Student identifies: "Simile"
Box 3: "The classroom was a zoo." → Student identifies: "Metaphor"
```

#### Game 3E: Spaced Flashcards

**Research:** Spaced Repetition + Forgetting Curve (Ebbinghaus, 1885) - 50-70% better retention with spaced review.

**STANDARDS:**

| Element | Specification |
|---------|---------------|
| **Card Count** | 5-10 cards per session |
| **Spacing Algorithm** | Leitner System or SM-2 |
| **Timer** | 2-3 minutes |
| **XP Award** | 50 XP per correct recall |

**Spacing Intervals:**

| Performance | Next Review |
|-------------|-------------|
| Correct on 1st try | 3 days |
| Correct on 2nd try | 2 days |
| Correct on 3rd try | 1 day |
| Incorrect | Review again today |

**Subject Examples:**

**Vocabulary (English as Foreign Language):**
```
Front: "Ancient" (Uzbek: "Qadimgi")
Back: "Very old, from long ago"
Example: "The ancient city of Samarkand is 2,700 years old."
```

**Science (Biology):**
```
Front: "Photosynthesis"
Back: "Process by which plants make food using sunlight"
Formula: "6CO₂ + 6H₂O + sunlight → C₆H₁₂O₆ + 6O₂"
```

### 2.7 Step 6: Final Boss (5-10 minutes)

**Purpose:** Mastery gate - assess deep understanding under productive struggle

**Research Basis:** Japanese Productive Struggle (Stigler & Hiebert, 1999) - 2+ PISA sub-level increase within 6 months.

**CRITICAL:** This is a **MANDATORY GATE**. Students cannot proceed without defeating the boss.

**STANDARDS:**

| Element | Grades 1-4 | Grades 5-8 | Grades 9-11 |
|---------|-----------|-----------|-------------|
| **Boss HP** | 50 HP | 100 HP | 150 HP |
| **Question Count** | 5 questions | 5 questions | 5-7 questions |
| **Easy Answer Damage** | -10 HP | -10 HP | -10 HP |
| **Medium Answer Damage** | -20 HP | -20 HP | -20 HP |
| **Hard Answer Damage** | -30 HP | -30 HP | -30 HP |
| **Combo (3 correct)** | 2x damage | 2x damage | 2x damage |
| **Hint Cost** | -10 HP (boss heals) | -10 HP | -10 HP |
| **Question Type** | Multiple choice | Mixed | Open reasoning |
| **Time Limit** | None | None | None |

**Victory Conditions:**

| Stars | Requirement | Unlocks |
|-------|-------------|---------|
| ⭐ | Complete boss (any attempts) | Chapter complete |
| ⭐⭐ | <3 attempts, >50% HP remaining | Bonus avatar item |
| ⭐⭐⭐ | 1st attempt, >80% HP, <5 min | Special badge progress |

**Defeat Consequences:**

```
┌─────────────────────────────────────┐
│  ❌ BOSS DEFEATED YOU               │
```

### 10.5 Game 5: Adaptive Quiz - Full Specification

**Purpose:** Precise ability measurement through IRT-based adaptation

**Complete Mechanics:**
```
ITEM RESPONSE THEORY (IRT) IMPLEMENTATION:

Question Bank Requirements:
  - Each question pre-calibrated with:
    - difficulty (θ): -3.0 to +3.0
    - discrimination (α): 0.5 to 2.0
    - guessing (c): 0.1 to 0.25

Adaptation Algorithm:
  1. Start with θ = 0 (grade-level expectation)
  
  2. Present question with difficulty ≈ current θ
  
  3. Student responds:
     IF correct:
       θ += 0.3 (increase estimated ability)
     IF incorrect:
       θ -= 0.3 (decrease estimated ability)
  
  4. Select next question with difficulty ≈ new θ
  
  5. Repeat for 5-7 questions
  
  6. Final θ = student's ability estimate

ABILITY INTERPRETATION:
  θ < -2.0: Significantly below grade level
  θ -1.0 to -2.0: Below grade level
  θ -0.5 to +0.5: At grade level
  θ +0.5 to +1.5: Above grade level
  θ > +1.5: Significantly above grade level
```

### 10.13 Game 13: Final Boss - Full Specification

**Purpose:** Mastery assessment under productive struggle

**Complete Mechanics:**
```
BOSS BATTLE FLOW:

INITIALIZATION:
  - Display boss avatar and HP bar
  - Show current combo counter (empty)
  - Display hint counter (if any)

QUESTION PRESENTATION:
  For each of 5-7 questions:
    1. Present question (text + visual if applicable)
    2. Student submits answer
    3. Evaluate:
       IF correct:
         - Calculate damage: 10/20/30 HP based on difficulty
         - Apply combo multiplier (if 3+ streak)
         - Update HP bar
         - Increase combo counter
         - Play damage sound
       IF incorrect:
         - Reset combo counter
         - Boss heals 5 HP (visual: green flash)
         - Play "miss" sound

HINT SYSTEM:
  - Request hint → Boss heals 10 HP
  - Hint displayed
  - Student answers (can still deal damage)

VICTORY:
  - Boss HP ≤ 0
  - Calculate stars based on:
    - Attempts: 1st, 2nd, or 3rd
    - HP remaining: >80%, >50%, or <50%
    - Time: <5 min, <8 min, or any
  - Award XP and stars

DEFEAT:
  - All questions answered, boss HP > 0
  - OR: 3 attempts failed
  - Route to remediation
```

### 10.14 Game 14: Memory Palace - Full Specification

**Purpose:** Long-term retention through spatial mnemonics

**Complete Mechanics:**
```
PALACE CONSTRUCTION:

Step 1 - Select Palace (30 seconds):
  Options:
    - Registan Square (default)
    - Student's home (personalized)
    - School building
    - Local bazaar

Step 2 - Place Concepts (60 seconds):
  For each concept (5 total):
    - Display concept name
    - Student selects location from 5 options
    - Student visualizes connection (10 seconds)

Step 3 - Mental Walkthrough (30 seconds):
  - Guide student through palace
  - "See" each concept at its location
  - Reinforce visual-spatial connection

Step 4 - Recall Test (60 seconds):
  - "What was at the Main Gate?"
  - "Which concept was by the Fountain?"
  - 5 questions total

SCORING:
  - Each correct recall: +50 XP
  - All 5 correct: +50 bonus XP
  - Total: 300 XP maximum
```
