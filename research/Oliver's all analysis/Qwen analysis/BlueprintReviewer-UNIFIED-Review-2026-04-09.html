Critical Findings

Finding 1: Version Number Inconsistency Low

Location: UNIFIED.md header vs. QWEN.md

The UNIFIED spec declares itself as Version 2.0 — April 2, 2026, but contains multiple "UPDATED 2026-04-07" annotations throughout.

Recommendation: Update UNIFIED.md header to Version 2.1 — April 7, 2026 to reflect the substantial additions.

Finding 2: Session Duration Contradiction (Still Unresolved) Medium

Location: §4.1 vs. multiple sections

Section 4.1 declares Standard (Homework) as 30-45 minutes, but Pre-Homework (0-A + 0-B) adds 3-5 min, pushing total to 28-44 min. This is tight for the lower bound.

Recommendation:

  1. Clarify whether Pre-Homework is included in the 30-45 min target or additional to it.
  2. If additional, adjust Standard duration to 35-50 minutes.

Finding 3: History Dual-Domain PISA Mapping — Implementation Gap Medium

Location: §7.4

Section 7.4 states that History uses dual-domain PISA mapping (Reading + Creative Thinking), but the student PISA profile structure (§3.3) shows no history domain or dual-domain structure.

Recommendation: Add history to the student profile JSON structure with dual-domain tracking and conversion logic.

Finding 4: Notebook Capture — Privacy Compliance Gap High

Location: §6.8

Issues identified:

  • No explicit parental consent mechanism for students under 13/16
  • No data sovereignty statement for Uzbekistan
  • Client-side blurring insufficient — EXIF data and metadata must be stripped

Recommendation: Add parental consent flow, data sovereignty clause, and metadata stripping requirement.

Finding 5: Mystery Box Reward Pool — Gambling Concern Low-Medium

Location: §6.3

The reward pool includes Mythical Boss Event Ticket (3% rarity) — a gacha-style mechanic that mirrors loot box mechanics.

Recommendation: Add disclosure to parents, consider replacing with guaranteed progress mechanic after 33 attempts.

Medium-Priority Findings

Finding 6: Game Selection Algorithm — Computational Complexity Medium

Location: §5.3

The Phase 3 Game Selection Algorithm must solve a constraint satisfaction problem that could be computationally expensive on low-end devices.

Recommendation: Pre-compute valid game combinations offline (Tier 1 processing). Runtime becomes simple lookup + filtering.

Finding 7: AI Tier Cost Modeling — Missing Projections Medium

Location: QWEN.md states "$3-5/student/year at national scale"

The spec does not provide per-request cost estimates, throughput calculations, or scaling model.

Recommendation: Add cost modeling appendix with per-tier projections.

Finding 8: Boss Retry Limit — Still Unresolved Low

Location: QWEN.md "Known Open Decisions" #1

UNIFIED spec §5.6 has no explicit retry limit, but §6.6 states "no upper limit on attempts." This creates ambiguity.

Recommendation: Add explicit retry limit to §5.6: "Maximum 2 boss re-attempts per session."

Low-Priority Findings

Finding 9: Missing Subject Frameworks Low

Current State: 5 of 9 subjects have frameworks. Missing: Tarix, Matematika, Ona tili, Ingliz tili.

Recommendation: Prioritize Matematika and Ona tili frameworks next (core PISA-assessed subjects).

Finding 10: Scholar Pairing Rule — Not Yet Implemented Low

Location: QWEN.md "Known Open Decisions" #6

The spec mentions cultural narratives but does not mandate inclusion of Central Asian scholars.

Recommendation: Add scholar pairing mandate to §5.2 (Story Mode).
