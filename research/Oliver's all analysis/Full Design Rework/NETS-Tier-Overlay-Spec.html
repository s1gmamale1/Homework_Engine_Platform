00 Design Principle

Both Basic and Premium are paid tiers. There is no free tier.

BASIC delivers the full product — everything in UNIFIED v2.0 and all subject frameworks. Complete, gamified, AI-supported, full national curriculum.

PREMIUM delivers everything in Basic plus more — more games, more resources, harder difficulty, richer AI, specialized courses. Premium is enrichment, not the "real" product.

The test: If removing Premium would leave Basic students with an incomplete education, the tier split is wrong. Basic alone must be sufficient to hit the PISA mission targets.

02 Detailed Specifications

Click any dimension to expand details.

2.1 Content

▼
Basic
Every chapter, topic, and standard in the Uzbek textbook canon. 100% curriculum coverage. All content tagged with dual codes (UZ-MATH-5-FRAC-01 / MAT.5.3.4.1), PISA levels, and transition_skill tags.
Premium adds
  • Beyond-textbook extensions — deeper explorations past textbook scope (e.g. Solar System → space exploration module with L4-L5 tasks)
  • Exam preparation tracks — structured revision calibrated to current PISA level
  • Elective modules — cross-subject enrichment (see Specialized Courses)
Rule: No curriculum-required content is Premium-exclusive. If the textbook teaches it, Basic has it.

2.2 Homework Sessions

▼
Basic
Full 7-phase session (0-A, 0-B, 1-7). All session parameters per subject framework: duration, flow targets, mechanic bans, Notebook Capture frequency, Boss HP tiers, Why Chain caps.
Premium adds
  • Expanded difficulty band — L5-L6 stretch tasks after consistent 85%+ scores. Basic caps at current+1; Premium allows current+2
  • Richer per-phase AI feedback — Tier 3 narrative branching in Story Mode, connection maps in Consolidation, personalized Reflection summary
  • Session replay — replay completed sessions with AI commentary on strengths/weaknesses
Rule: 7-phase structure, immutables (Stranger Test, transition_skill, cosmetics rule, 0-A/0-B), and all subject framework overrides apply identically to both tiers.

2.3 Games

▼
Basic
Phase 3 Game Breaks every session: 3 games (≥1 Interactive Catalog + ≥2 Default Pool), selected by subject + Bloom's + PISA level. All 28 mechanics available per subject framework.
Premium adds
  • On-demand game access — launch any unlocked game from Library anytime outside sessions
  • Higher-polish variants — enhanced visuals/audio, same gameplay and learning outcomes
  • Exclusive game modes — timed challenges, head-to-head, Master Mode
Rule: Within a session, both tiers get the same Phase 3. Dual-catalog rule is never violated. Premium's advantage is access outside sessions.

2.4 AI Tier

▼
Basic — $3-5/student/year
Tier 1 (rule-based, ~$0) handles ~80% of tasks. Tier 2 (small LLM) handles Boss evaluation (2 attempts max), anti-cheat, story checkpoints. Fail Boss twice → marked as failed, next session continues.
Premium adds — $8-15/student/year est.
  • Personal AI Tutor on Boss failure — Tier 3 Socratic tutor activates after 2 fails. Guides student through with dialogue (no answer reveal) + question regeneration. Unlimited retries until Boss is defeated. Like having a personal teacher.
  • Tier 3 on any question — ask "why?" anywhere, get guided dialogue
  • Personalized cross-subject study plans
  • Detailed writing feedback
  • Exam prep adaptation
Rule: Tier 2 anti-cheat runs for both tiers. Fallback: T3/T2 unreachable → T1 takes over. No session ever fails because AI is unavailable.

2.5 Notebook Capture

▼
Basic
Frequency per subject framework (Science 1-in-3, Texnologiya 1-in-2, Tasviriy Sanat every session, Geografiya 1-in-4). Tier 2 evaluation: pass/fail + one tip.
Premium adds
  • Tier 3 vision feedback — detailed rubric (diagram accuracy, technique analysis, safety compliance)
  • Portfolio — AI-curated gallery of best captures with progression commentary
Rule: Frequency is NOT tier-gated. If the framework says 1-in-3, both tiers do 1-in-3. Difference is feedback depth.

2.7 PISA Events

▼
Basic
Mandatory weekly PISA event — ~20 min, 7-phase session with PISA-style content drawn from active subjects. Delivered within a 7-day async rolling window. Teacher dashboard tracks completion. Required for every student regardless of tier.
Premium adds
  • 2 additional PISA events per week (3× weekly total) — same format, more practice volume
  • Strict-deadline enforcement (future option, not launch default)
Rule: Weekly PISA event is mandatory for both tiers — PISA preparation is the mission, not a paywalled feature. Premium only adds frequency.

2.8 Metagame (Bilim Bazasi)

▼
Basic
Full Bilim Bazasi: visual base, BT earning (sessions/skills/Bosses/streaks), hero unlocks (Al-Khwarizmi, Ibn Sina, Ulugh Beg, Beruniy), Mystery Boxes (BT-only), streaks with earned freeze, class tournaments.
Premium adds
  • Faster BT earning in tournaments (multiplier TBD via playtesting)
  • Additional cosmetic collections, seasonal sets
  • Seasonal events with unique cosmetic rewards
  • School-level and regional tournament tiers (opt-in)
Rule: BT has no real-money entry. No academic advantage purchasable. Basic base can grow just as large — Premium has more decorating options.

2.9 Session Modes

▼
Basic
Standard (30-45 min) + Recovery (15-20 min, auto-triggered). Extended (45-60 min) via teacher override only.
Premium adds
  • Extended on-demand — 45-60 min sessions at home without teacher override

07 Open Questions

1Premium pricing model — family-paid, school-licensed, government-sponsored, corporate, or hybrid.
2BT tournament multiplier — how much faster? Too high = pay-to-win violation. Needs playtesting.
3Premium cost validation — $8-15 estimate assumes 2-3 Tier 3 calls/session. Needs real usage data.
4Specialized course content pipeline — no textbook source. Who writes these? Quality bar?
5Upsell localization — Uzbek + Russian. Culturally appropriate tone.
6Extended mode learning impact — 45-60 min at home risks attention fatigue. Break prompt at 35 min?
7RESOLVED — Tier 3 Boss failure — Basic: 2 attempts, fail = marked, next session. Premium: T3 personal tutor with unlimited retries. UNIFIED needs a versioned note for this Layer 3 override.
