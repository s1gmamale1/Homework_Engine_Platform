01 The NETS Learning Pyramid

Every homework session MUST map to Bloom's Taxonomy through the NETS Learning Pyramid. Sessions must include activities spanning at least 4 levels.

TRANSFER PISA L5-6 Real-Life Extended
ANALYZE PISA L4 Why Chain, Mystery Box
APPLY PISA L3 Adaptive Quiz, Final Boss
UNDERSTAND PISA L2 Story Mode, Tile Match
REMEMBER PISA L1 Memory Sprint, Flashcards
ENGAGE (Foundation) All Levels Gamification, Flow State
NON-NEGOTIABLE: Every homework session MUST include activities spanning at least 4 levels of the pyramid. The ENGAGE layer is always active as the foundation.

02 The 3 Sacred Principles

📚
Textbook is Source of Truth
NETS transforms engagement with textbook content — it never replaces it. Homework MUST NOT change core concepts. It adapts delivery to active, gamified, AI-personalized learning.
Textbook providesCore concepts
NETS providesActive reasoning
ResultDeep engagement
🎯
Flow State is Mandatory
Based on Csikszentmihalyi (1990). Every student MUST be maintained in the flow channel: 70-80% success rate. AI adjusts difficulty every 3-5 questions.
< 60% successReduce difficulty -1 tier
70-80% successFLOW ZONE — Maintain
> 90% successIncrease difficulty +1 tier
🏆
Intrinsic Motivation > Extrinsic
All rewards celebrate LEARNING PROGRESS, not compliance. Zero pay-to-win. Zero real-money purchases. Every reward is earned through demonstrated learning.
DO: "You mastered fractions!"
DO: "Your reasoning improved!"
DON'T: "You logged in 7 days!"
DON'T: "You watched an ad!"
"No Busywork" Rule: Every task must explicitly scaffold ONE specific PISA transition skill. No filler questions. No repetition without purpose. If a task cannot be tagged to a transition skill, it does not enter the content pool.

20 Edge Cases and Recovery Modes

Daily realities for 8.8M students. Each scenario has a defined response.

ScenarioTriggerNETS Response
Missed Homework Within 3-day grace period Recovery session (15-20 min). Extended Memory Sprint, condensed Story Mode, 1 game break, full Final Boss. No shortcuts on assessment.
Multiple Missed >1 session missed Sort by dependency, assign in order. Max 2 recovery sessions/day. Teacher sees queue + estimated completion.
Extended Absence >7 days absent Catch-Up Mode: essential concepts only, deferred non-essentials, Boss HP reduced 50%, hints free. Max 2 sessions/day.
Falling Behind PISA level < class avg - 0.5 for 3+ sessions Boost Mode: targeted practice, bonus XP incentives, AI improvement plan, parent + teacher notified.
