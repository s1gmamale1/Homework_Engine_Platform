Post-Session: Ma'lumotlar saqlash va feed-forward

1. PISA profil yangilash
Yangi natijalar bilan PISA daraja qayta hisoblash, domen taqsimotini qayta hisoblash
2. Spaced repetition
SM-2 algoritmi: noto'g'ri → interval=0. To'g'ri → intervalni uzaytirish. Yangi → 1 kundan keyin.
3. Mastery xaritasi
Ebbinghaus so'nish: mastery vaqt o'tishi bilan pasayadi. <0.6 → Memory Sprint ga kiritish
4. O'tish ko'nikmalari
>80% aniqlik 3+ sessiyada → "mastered". Yangi nishonlarni aniqlash.
5. Traektoriya tekshirish
3 ketma-ket sessiya tushish → BOOST MODE. Sinf maqsadidan oshish → boyitish kontenti.
