B. Banned Techniques

  • ❌ Public displays of wrong answers — fragile self-concept; private correction only
  • ❌ Absolute-rank leaderboards — only improvement ranking or class-wide goals
  • ❌ Real-time multiplayer competition — use async ("beat your own score") instead
  • ❌ Pure discovery / unbounded sandbox — always provide guided structure
  • ❌ Random rewards without competence link — opportunity must be earned
  • ❌ Purchasable cosmetics — all avatar items earned through competence
  • ❌ Abstract-only puzzles — CPA mandatory for Science
  • ❌ Complex multi-step instructions without worked examples
  • ❌ Delayed feedback — all feedback within seconds
  • ❌ "You're smart" / ability praise — use effort + strategy praise
  • ❌ The word "wrong" — use "not yet" / "not quite" / "close — here's why"
  • ❌ Brain breaks every 5 min — phase transitions ARE the breaks
  • ❌ Learning styles matching — no evidence; ignore
  • ❌ Why Chain ≥3 levels — over WM ceiling for G5
  • ❌ Visible difficulty labels — triggers self-handicapping; hidden from UI
  • ❌ Group/peer-correction as primary feedback — solo-first for concept acquisition

C. Mandatory Additions

  • ✅ CPA progression — Concrete → Pictorial → Abstract for every Science concept
  • ✅ Dual coding — text paired WITH image, never text-only
  • ✅ Worked example before independent practice
  • ✅ Spaced retrieval — Day 1 / Day 3 / Day 7 / Day 14 schedule
  • ✅ Effort + strategy feedback — "You found a strategy that didn't work — let's try another"
  • ✅ Uzbek cultural anchors — Ulugh Beg observatory, Aral Sea, Registan, Amir Temur
  • ✅ Mobile-first, portrait orientation, low-bandwidth
  • ✅ Variable-ratio reward windows — earned via competence
  • ✅ Hidden difficulty tiering — adapts silently
  • ✅ 30-second productive struggle window — before offering hint

🔬 1. Why a Science-Specific Engine

  • Triple-domain textbook — biology (Ch.1–2), physics/chemistry (Ch.3–7), earth science/geography (Ch.8–10). One Story Mode template can't serve all three.
  • PISA Science L1→L2 transition is the national bottleneck — baseline 355, target L2. Every session must scaffold this transition explicitly.
  • Textbook is inquiry-based — chapters open with O'ylab ko'ring (think) and end with Bilimingizni sinab ko'ring (test).
  • Tadqiqotchilik ko'nikmalari (research/inquiry skills) is a transversal competency tagged on every chapter.

🔒 2. Inherited Immutables (No Override)

  • Pre-homework Sessions 0-A (Theme Preview) + 0-B (Flash Cards) — student-paced, no scoring, no XP, returnable; timer starts only after "Start my Homework" tap
  • 7 phases + 16 game mechanics (Memory Palace #5, Tic-Tac-Toe vs AI #15, Notebook Capture #16 — per UNIFIED 2026-04-07)

  • Boss is mandatory, tiered (Sub / Big / Mythical)
  • Textbook-as-source-of-truth, Stranger Test on Story Mode
  • Mandatory tags: textbook_ref, standard_ref, pisa_ref (with transition_skill), blooms_level, game_mechanic

Session 0-B — Flash Cards

Duration1–2 min initial pass; returnable throughout homework via "Flash Cards" button (NOT a hint — pure reference)
FormatSwipeable card pool, one concept per card, no testing
TierAI Tier 1 (pre-generated per chapter)

Card Pool Must Contain (per UNIFIED §4.5):

  • All Key Formulas — density (Bob 4), force diagrams (Bob 5)
  • All Key Concepts — magnet poles, adaptation, water cycle stages
  • All Key Rules — like poles repel, opposite attract; predator-prey balance

Card structure: Front = concept name + textbook page; Back = 1–2 sentence definition + image/diagram

Ends with prominent "Boshlash" / "Start my Homework" button. Only after this button does the timer start and XP accrue.

Why These Matter for Science G5

  • Vocabulary scaffold — 0-B preloads all new technical terms, satisfying the §0.A "max 1–2 new terms" rule
  • Cultural priming — 0-A's Uzbek anchors lock in cultural relevance before cognitive load begins
  • Anxiety buffer — student-paced exploration with no penalty respects fragile G5 self-concept
  • Fairness for slow readers — Lexile-controlled, swipeable, no time pressure

🎮 5. Phase-by-Phase Science Overrides

Phase 1 — Memory Sprint (≤2 min)

  • Recall items pull from Ilmiy atamalar (glossary) + prior chapter terminology
  • Format: 50% MC term-definition, 30% Tile Match, 20% Sentence Fill
  • No abstract calculation — belongs in Game Breaks
  • Remediation gate: <60% → route to remediation chapter

Phase 2 — Story Mode (5–7 min)

Template:

Phenomenon  →  Predict  →  Observe/Results  →  Explain

This mirrors the textbook's own O'ylab ko'ring → activity → explanation pattern.

DomainPhenomenon HookPredict GateResults GateExplain Gate
Biology (Ch.1-2)Time-lapse / image"What will happen if…?" MCAnimation or photoFree-text or scaffolded
Phys/Chem (Ch.3-7)Demo videoPredict outcomeShow actual resultTile-Match cause→effect
Earth (Ch.8-9)Satellite image / weatherPredict patternReveal dataWhy-Chain (2 levels)

Phase 3 — Game Breaks (6–9 min)

Game MechanicWhen to UseBloom'sPISA

Selection algorithm (3 games): 1) Reinforcement game at current Bloom's → 2) Stretch game one level above → 3) Transition-skill game scaffolding L1→L2 or L2→L3.

Phase 4 — Real-Life Challenge (3–5 min)

  • First-person expert POV mandatory — Uzbek-context roles: farmer in Andijon, teacher in Tashkent, meteorologist, environmental inspector at Aral Sea
  • Multi-path reasoning required (≥2 valid approaches)
  • Inquiry hook: ends with "What would you measure?" / "What would your control group be?"

Phase 5 — Consolidation / Mnemonic Lock (3–5 min)

Default technique: Memory Palace (mechanic #5) using culturally-anchored Uzbek locations. 5 loci = 5 concepts from session, each with concept + visual + recall question.

Structure

  1. 5 loci mapped to 5 concepts from current chapter
  2. Each locus card: location image + concept text + visual mnemonic + 1 recall question
  3. Recall test at end: 3 questions cycling through loci, no hints
  4. Interleaving: 1 of 5 loci references a prior-chapter concept

Alternative: Adaptive Quiz (only if Memory Palace was used in prior session — avoid back-to-back same mechanic).

Phase 6 — Final Boss

Student PISABoss TierHP / PatternExpected Outcome
Below L1 (≈355)Sub Boss Easy30 HP, L1 onlyRequired win, scaffolded
L1–L2 (median G5)Sub Boss Medium50 HP, L1–L2 mixWin after 1 retry
L2–L3 (top quartile)Big Boss (weekly) or Sub Boss Hard80 HP, L2–L3Real challenge
L3+ (top decile)Big Boss + occasional Mythical100+ HP, L3–L4Stretch

Boss Content Requirements

  • 5 questions, distribution: Easy 40% / Medium 40% / Hard 20%
  • 1 phenomenon-explanation (Tier 3 AI evaluates reasoning)
  • 1 data-interpretation (chart, table, or diagram from textbook)
  • 1 cause-effect chain question
  • 1 vocabulary/recall (anchors L1)
  • Optional Hard: 1 inquiry-design — Sub Boss Hard / Big Boss only

Each Boss Question MUST Carry

  • Step-by-step worked solution (visible after answer)
  • 2–3 progressive Hints (revealed on request, not auto-pushed)
  • Common Errors list — at least 1 known wrong answer with diagnostic explanation
HIDDEN Difficulty labels ([EASY] / [MEDIUM] / [HARD]) are INTERNAL ONLY — never shown to student in UI (per §0.B.15 hidden difficulty rule).

Mastery Stars (1–3) — Earned Per Session

StarsCondition
⭐ 1Boss defeated (any attempts within retry limit)
⭐⭐ 2Boss defeated in ≤2 attempts AND >50% HP remaining
⭐⭐⭐ 3Boss defeated on FIRST attempt, no hints used, >80% HP remaining

Failure → Boss Regeneration

If student fails all permitted retries, Tier 3 AI runs Socratic tutoring based on wrong answers, then regenerates new boss questions tailored to the gap. Auto-flags teacher dashboard after 3 failed sessions on the same standard.

Boss is mandatory. Max 2 retries per session; on 2× fail → next session continues from boss with simplified scaffold + Tier 3 Socratic intro (UNIFIED §5.6).

Universal Science Prompts (rotate when not bucket-overridden)

  • "Bugun qaysi tabiat hodisasini yangidan tushundingiz?"
  • "Tajribangizda nima kutilganidan boshqacha bo'ldi?"
  • "Bu mavzu kundalik hayotingizda qayerda uchraydi?"

Rules

  • Min 10 characters (teacher configurable up to 5 sentences)
  • Bloom's = Create, PISA = L3-4 (metacognitive only)
  • Effort + strategy language only — never "Were you smart enough?"
  • Optional peer-explanation: "Bu mavzuni darsda yo'q bo'lgan do'stingizga qanday tushuntirgan bo'lardingiz?"

60% minimum to pass session (G5 override §0). Below 60% → Duolingo Mode triggers automatically.
