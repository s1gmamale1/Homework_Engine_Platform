## 7-Step Learning Journey (20-30 min total)

```
┌─────────┬──────────────┬─────────┬────────────────────────────────┐
│ Step    │ Name         │ Time    │ Purpose                        │
├─────────┼──────────────┼─────────┼────────────────────────────────┤
│ 1       │ Memory Sprint│ 2 min   │ Activate prior knowledge       │
│ 2       │ Story Mode   │ 5-7 min │ Deliver new content (narrative)│
│ 3       │ Game Breaks  │ 6-9 min │ Reinforce via gamified practice│
│ 4       │ Real-Life    │ 3-5 min │ Transfer to authentic contexts │
│ 5       │ Consolidation│ 2-3 min │ Lock in learning (mnemonics)   │
│ 6       │ Final Boss   │ 5-10 min│ Mastery gate (must defeat)     │
│ 7       │ Reflection   │ 2-3 min │ Build metacognition            │
└─────────┴──────────────┴─────────┴────────────────────────────────┘
```

**CRITICAL:** Step 6 (Final Boss) is a MANDATORY gate. No exceptions.

## Final Boss Rules

### HP System
| Grade | Boss HP | Easy Dmg | Med Dmg | Hard Dmg |
|-------|---------|----------|---------|----------|
| 1-4   | 50 HP   | -10 HP   | -20 HP  | -30 HP   |
| 5-8   | 100 HP  | -10 HP   | -20 HP  | -30 HP   |
| 9-11  | 150 HP  | -10 HP   | -20 HP  | -30 HP   |

### Combo System
- 3 correct in a row → 2x damage on next answer
- Combo resets on wrong answer

### Hint Tax
- Requesting hint → Boss regains 10 HP
- No hints on first attempt (Grades 5+)

### Victory Conditions
| Stars | Requirement | Unlocks |
|-------|-------------|---------|
| ⭐ | Complete boss | Chapter complete |
| ⭐⭐ | <3 attempts, >50% HP | Bonus avatar item |
| ⭐⭐⭐ | 1st attempt, >80% HP, <5min | Special badge progress |

### Defeat Flow
1. Boss not defeated → Route to specific remediation section
2. AI becomes Socratic tutor (guiding questions, no answers)
3. Student retries with new questions (same concepts)

## Resilience Systems

### Recovery Queue (1-3 days absent)
- Grace period: 3 days (teacher configurable)
- Condensed sessions (remove warm-up)
- Extended Memory Sprint (decay compensation)

### Catch-Up Mode (1+ weeks absent)
- Accelerated path (essential concepts only)
- No non-critical game breaks
- Simplified Final Boss (reduced HP, hints allowed without penalty)

### Boost Mode (falling behind)
- Trigger: PISA level < class average × 2 weeks
- Targeted practice sessions (optional, incentivized)
- Bonus XP + special avatar items

### Low Engagement Intervention
- Trigger: <50% completion × 2 weeks
- Response:
  1. Reduce session length
  2. Increase game-to-content ratio
  3. Parent notification with specific suggestions
  4. Teacher dashboard flag

## UI/UX Key Principles

### Colors
```
Primary:    #6366F1 (Indigo)
Success:    #22C55E (Green)
Warning:    #F59E0B (Amber)
Danger:     #EF4444 (Red)
Background: #0F172A (Slate 900)
Surface:    #1E293B (Slate 800)
```

### Interaction Rules
1. **Immediate feedback** (<100ms response)
2. **Progressive disclosure** (show only what's needed now)
3. **Error prevention** (confirm destructive actions)
4. **Accessibility** (44px touch targets, high contrast, screen reader)

### Animation Guidelines
- Duration: 200-300ms
- Easing: ease-in-out
- Purpose: Feedback, orientation, delight (never decorative only)
- Respect: Reduced motion option available
