# Prompt: Preview — Kimyo — Hard Mode

You are building the Preview phase for a Kimyo homework session (Hard mode). You will receive a textbook page. Your job is to create 7 teaching panels in Uzbek that fully prepare the student to reason across all three scales (macroscopic → microscopic → symbolic), balance chemical equations, apply the Periodic Table as a prediction tool, and work safely in a lab context.

This is Hard mode — the student will face a Real-Life Challenge later where they must observe a substance, reason at all three scales, write a balanced equation, and solve a stoichiometric or safety problem. Panel 5 (Observable→Formula Translation) is the critical bridge. If you don't teach them HOW to move from what they observe to what they write, the Real-Life phase becomes guesswork.

## Input

- Textbook page (image or text)
- Grade: G7-11 (Kimyo)

## Output

7 panels in order. Each panel in Uzbek, formal "Siz" throughout. Never "sen."

## Panel 4: Examples

Each example uses **progressive scale states** — the description builds across all three scales step by step. Do not show a finished equation; show the reasoning being constructed:

```
[State 0 — Safety: identify hazard and PPE required before handling this substance]
[State 1 — Macro: describe what you observe — color, state, observable reaction signs]
```

## Panel 5: Observable → Formula Translation

THIS IS THE BRIDGE TO REAL-LIFE. Without this panel, the Real-Life Challenge becomes guesswork.

Teach the student HOW to:
1. **Observe** macroscopic properties of a substance (color, state, smell, reactivity) and describe them precisely
2. **Identify** what type of substance it is from observation alone — metal/nonmetal, acid/base, oxide, salt
3. **Construct** the correct symbolic formula using valence rules or Periodic Table position
4. **Write** and balance the chemical equation for the reaction, verifying atom counts

Structure:
- 2-3 mini-cases, building in complexity
- Each mini-case: safety note → macroscopic observation → substance identification → microscopic model → symbolic formula or equation → balance verification
- First case: simple — one substance, observable properties directly confirm the formula
- Last case: chapter-level complexity — with boundary observations (what macroscopic observations are NOT sufficient to determine the formula uniquely; student must apply Periodic Table or valence to resolve ambiguity)

Example pattern:
> `[Safety: corrosive liquid — goggles and gloves required (red)]`
> Macro observation: rangsiz suyuqlik, o'tkir hidli, ko'k lakmusni qizartiradi, metallarni eritadi.
> Identification: kislota xossalari — H⁺ ionlari mavjud.
> Micro model: H⁺ va Cl⁻ ionlari suvda dissotsilanadi.
> Symbolic: HCl (xlorid kislota) — H valentligi +1, Cl valentligi -1.
> Balance check: HCl + NaOH → NaCl + H₂O → H×1=H×1, Cl×1=Cl×1, Na×1=Na×1, O×1=O×1 ✓

- Teach the boundary cases: what additional information is needed when observable properties alone are insufficient to identify the formula?
- 10-12 sentences total across mini-cases
