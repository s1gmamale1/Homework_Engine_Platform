# Prompt: Real-Life Challenge — Kimyo (Hard only)

You are building the Real-Life Challenge (Phase 4) for a Kimyo Hard mode session. The student has completed Preview, Flash Cards, Sprint, and Game Breaks. Now they apply everything to ONE deep real-world scenario as THE EXPERT.

This phase tests whether Preview's Observable→Formula Translation teaching actually worked. The student must observe a substance, identify all three scales, write the balanced equation, state safety precautions, and solve a stoichiometric or contextual chemistry problem.

## Input

- Textbook page (image or text)
- All previous phase outputs
- Grade: G7-11 (Kimyo)

## Output

ONE scenario with 4-6 sub-questions.

---

## Scenario Construction

**First-person POV mandatory.** The student IS the expert.

> "Siz [professional role]. Sizning vazifangiz..."

Never third person ("Dilnoza needs to..."). Always direct: "Siz", "Sizning".

**Pick a professional role:**
- G7-11: kimyogar (chemist), farmatsevt (pharmacist), ekolog (environmental scientist), oziq-ovqat texnologi (food technologist), neft-kimyo muhandisi (petroleum chemist), agrokimyogar (agronomist), tibbiy laborant (medical lab technician)

**Scenario structure:**
1. Safety preamble (1-2 sentences) — state the hazard and required PPE before any scenario content
2. Context narrative (3-4 sentences) — modern professional Uzbek setting with a substance identification or reaction problem embedded
3. A three-scale reference description in brackets — the student must read this to extract the given information
4. Sub-questions (4-6) — each tests a skill from Preview

**Scope lock:** Every reaction, formula, and reasoning skill tested must have been taught in Preview. No new reaction types. Same difficulty as the hardest Preview example — different context and numbers.

**Context policy:** Modern professional only. Navoiy kimyo kombinati, Farg'ona neftni qayta ishlash zavodi, Toshkent farmatsevtika klasterlari, O'zbekiston qishloq xo'jaligi, Orol dengizi ekologik loyihalari, tibbiy klinikalar, suv tozalash stansiyalari. NO bazaar, NO village, NO shopkeeper, NO farmer.

---

## Sub-question construction

Each sub-question:
- Requires reading the scenario three-scale description and identifying what is observed and what is known
- Requires identifying all three scales when relevant: "Uch darajada tasvirlang"
- Requires naming the reaction type before writing the equation: "Avval reaksiya turini ayting"
- Requires stating the relevant safety precaution when handling a hazardous substance
- Has one correct answer — balanced equation verified, or calculation with units shown

Mix:
- 1 safety question ("Bu modda bilan ishlashda qanday himoya vositalari kerak va nima uchun?")
- 1-2 three-scale reasoning questions ("Bu moddani uch darajada tasvirlang: makroskopik, mikroskopik, ramziy")
- 1-2 equation writing and balancing questions ("Reaksiya tenglamasini yozing va muvozanatlang — atom sonlarini tekshiring")
- 1 real-context calculation or prediction question (stoichiometric, or Periodic Table prediction)
- 1 interpretation question ("Bu natija mutaxassis sifatida Sizga nima anglatadi?")

---

## Diagram requirement

Every scenario MUST include a **two-state three-scale description** — not a single static entry. The description evolves as the student works through sub-questions:

**State A — Problem description (shown at the start):**
- The professional's situation: observable substance properties described (macro), substance type noted (to be identified), formula and equation to be written (orange "?"), safety status noted
- This is what the student reads to begin identification and reasoning

**State B — Annotated description (built during solving):**
- Student adds microscopic description as they reason through
- Student writes and verifies the balanced equation
- Student adds safety precautions
- Final state: all three scales complete, equation balanced, safety stated, calculation completed

Example:
> `[State A — Problem:
> Safety: unknown white powder — treat as potentially harmful, wear goggles and gloves until identified
> Macro: oq kristall kukun, hidsiz, suvda eriydi, eritmasi qizil lakmusni ko'k rangga o'zgartiradi (blue)
> Micro: ? (orange)
> Symbolic: formula ? (orange); reaksiya ? (orange)]`
>
> `[State B — After solving:
> Macro: oq kristall, ishqor xossalari
> Micro: Na⁺ va OH⁻ ionlari suvda
> Symbolic: NaOH (Na: +1, OH⁻: -1)
> Reaksiya: NaOH + HCl → NaCl + H₂O (balanced: Na×1=Na×1, O×1=O×1, H×2=H×2, Cl×1=Cl×1 ✓)
> Safety: kuydirgich — goggles + gloves majburiy]`

The scenario text tells the student: "Berilgan kuzatuvlardan moddani aniqlang — uch darajali tavsifni to'ldiring va reaksiya tenglamasini yozing."

## Rules

- ONE scenario only. Not multiple.
- First-person "Siz" POV. Never third person.
- Safety preamble BEFORE the scenario context — no exceptions
- Scenario MUST include a two-state three-scale description (State A = problem, State B = completed reasoning)
- Student MUST name the reaction type and state the safety rule before writing any equation
- All reaction types and formulas must come from Preview content — no new chemistry
- All balanced equations must include atom count verification
- Three-scale reasoning mandatory for substance identification questions — never one-scale answers
- Language: Uzbek, "Siz" formal
- No bazaar/village clichés
- Students instructed to annotate the three-scale description as they work: "Uch darajali tavsifni to'ldiring"
