# Context Policy (Modern Professional Context)

## Rule

All scenarios must place the student in the role of **a professional expert solving a real case**. Context must be modern, aspirational, and workplace-realistic. Cliché bazaar/village/shopkeeper/farmer scenarios are banned.

POV is always direct address: "You are..." / "Your job is..." / "A client comes to you...". Never third-person ("Tomir's problem is...").

## Scope

- **Primary:** Phase 4 Real-Life Challenge — mandatory for every scenario
- **Secondary:** Phase 2 Story Mode narrative framing and Phase 6 Final Boss challenge scenarios where context is set

## Banned terms / archetypes

The following scenario archetypes are prohibited:

- Bazaar / bozor / market stall seller
- Village / qishloq setting as primary frame
- Shopkeeper, farmer, shepherd
- "A boy named X is helping his grandmother..."
- Any scenario where the student is a passive observer rather than the decision-maker
