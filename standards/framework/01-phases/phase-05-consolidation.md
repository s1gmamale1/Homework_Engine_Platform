# Phase 5 — Consolidation / Mnemonic Lock

## Purpose (WHAT)

Mnemonic lock phase: re-encodes the lesson's key concepts from fragile short-term memory into durable long-term storage by routing them through a second cognitive channel (visual / spatial / associative), ensuring the memory trace is stored in multiple brain pathways rather than only the verbal channel used in Phase 2.

**Trigger rule (CONDITIONAL PHASE):** Phase 5 fires ONLY when the lesson introduces ≥2 interlocking concepts that require integration. It is SKIPPED when the lesson archetype is Concept Intro with a single standalone concept.

| Archetype | Phase 5 status |
|-----------|---------------|
| Algorithm | REQUIRED — ≥2 methods/steps demand method-choice synthesis |
| Application | REQUIRED — ≥2 concepts must be combined for the applied scenario |
| Review | REQUIRED — multi-concept review session always requires consolidation |
| Concept Intro (single concept) | SKIPPED — mnemonic lock on one concept has no integration value |
| Concept Intro (≥2 interlocking concepts) | REQUIRED |

## Structure (HOW)

**Standard mode (2–3 min):** 2 mnemonic exercises. Student attempts each once; unlimited retries allowed within the phase time budget.

**Extended mode (5 min):** 3 mnemonic exercises. The third is always a recall-test variant (items out of original order, or reverse-direction Peg recall).

**Student-facing flow:**

1. **Entry card** (3–5 sec): *"Endi bilganlaringizni mustahkamlaymiz."* (Now we lock in what you've learned.)
2. **Technique intro** (5–10 sec): brief explainer of the selected mnemonic technique with a worked example from the default Memory Palace (Registan Square) or grade-appropriate scene.
3. **Exercise 1** (45–75 sec): one mnemonic exercise using the content-structure-matched technique.
4. **Exercise 2** (45–75 sec): same technique with different items OR recall-tested out of original order.
5. **Transition** (3–5 sec): *"Yaxshi! Endi Boss uchun tayyormiz."* (Good! Now we're ready for the Boss.)

**Technique selection — content structure drives the choice, not random assignment:**

| Content Structure | Technique | Interaction Model | Grade restriction |
|------------------|-----------|-------------------|------------------|
| Hierarchical (categories, classifications) | **Radiant Summary** | Drag keywords onto partially completed mind map branches. Buzan's 7 Laws: central image, organic branches, one keyword per branch, 3+ colors. G5: max 4 branches (WM ceiling). | G5+ |
| Sequential (processes, timelines, life cycles) | **Link System** | 5-item vivid story chain; student orders items, then recalls reversed. SMASHIN' SCOPE imagery required. | G5+ |
| Spatial (locations, map features) | **Memory Palace** | 2.5D isometric scene (default: Registan Square), 5 hotspots, student places concepts at locations. | G3+ |
| Discrete facts (5 key terms, dates, vocabulary) | **Peg System** | Number-Shape pegs (1=Candle, 2=Swan, 3=Heart, 4=Sailboat, 5=Hook — Uzbek-adapted). Both directions tested (peg→concept and concept→peg). | G5+ |
| Numeric (dates, constants, formulas) | **Major System** | Phonetic digit code (1=T, 2=N, 3=M, 4=R…). Student reconstructs memorable word from digits. Example: 1441 → T-R-R-T → "Toshbaqa" (Turtle). | **G7+ only** |

**Grade-band availability:**

| Grade Band | Available Techniques |
|-----------|---------------------|
| 1–2 | Chunking, Rhymes, Songs |
| 3–4 | Memory Palace intro, Acronyms, Dual coding |
| 5–7 | Memory Palace (full Loci), Peg System, Radiant Summary, Link System |
| 7–9 | All G5–7 + Major System |
| 10–11 | All G7–9 + Self-directed systems, Autonomous consolidation |

**Subject-technique fit:**

| Subject Family | Best Techniques | Rarely Useful |
|---------------|----------------|--------------|
| Aniq Fanlar (Math) | Peg System (formula names), Radiant Summary (formula families) | Link System, Memory Palace |
| Tabiy Fanlar (Science) | Radiant Summary (classification), Link System (processes), Memory Palace (anatomy) | Major System |
| Ijtimoiy Fanlar (History/Geography) | Link System (timelines), Major System (dates), Memory Palace (locations), Radiant Summary (biomes) | — |
| Til Fanlar (Language) | Peg System (vocabulary sets), Link System (grammar rule chains) | Major System, Radiant Summary |

**Secondary delivery vehicles (game-flavored skins):**

- Memory Match Blitz (§6.18): allowed as a skin for Peg System exercises (5 discrete facts → pairing practice). Counts as one Phase 5 exercise.
- Reaction Chain (§6.19): allowed as a skin for Link System exercises when sequence has 5+ steps. Counts as one Phase 5 exercise.

**Banned mechanics in Phase 5:** Tile Match, Sentence Fill, Tic Tac Toe, Connect Four, Mystery Box, Puzzle Lock, Word Ladder, Creative Lab (Phase 3 mechanics). Adaptive Quiz, Story Mode, Real-Life Challenge (wrong-phase mechanics).

**Gate:** Formative only — no pass/fail. Student advances to Phase 6 after attempting both (Standard) or all three (Extended) exercises. Correctness is not required; attempt is sufficient.

**SMASHIN' SCOPE quality gate (content-pipeline, not student-facing):** Every Phase 5 exercise must score ≥6/12 on SMASHIN' SCOPE (Synaesthesia, Movement, Association, Substitution, Humor, Imagination, Number, Symbolism, Color, Order, Positive, Exaggeration). Content scoring below 6 is returned for enrichment before deployment. *(Threshold provisional pending pilot calibration.)*

**Failure / retry language (soft — never Phase-6 boss-tier language):**

| Situation | Uzbek | Russian |
|-----------|-------|---------|
| First attempt wrong | *"Yaqin edi — yana bir marta urinib ko'ring."* | *"Почти — попробуйте ещё раз."* |
| Second attempt wrong | *"Javobni ko'rsataymi?"* + reveal-then-retry | *"Показать ответ?"* |
| After reveal | *"Yaxshi. Keyingi mashqqa o'tamiz."* | *"Хорошо. Переходим к следующему."* |

Banned failure phrases in Phase 5: *"Hali emas!"*, *"Noto'g'ri"*, *"Xato"* — those are Phase 6 register only.

**Edge cases:**

- **Recovery Mode (§12.1):** student >7 days behind → Phase 5 collapses to 1-minute Quick Flashcard Review of critical `keywords_80_20` items only. Full mnemonic exercises skipped.
- **Offline Mode (§12.6):** fully offline-compatible — all exercises are Tier 1 pre-generated content, no live AI required.
- **Mythical Boss / Big Boss sessions:** Phase 5 is unchanged. These Phase 6 variants do not alter Phase 5 structure.

## Cognitive load

- **Bloom range:** L2 (Understand) → L3 (Apply). Not scored separately.
- **PISA range:** Not assessed. Phase 5 is formative rehearsal, not evaluation.
- **Scoring:** None. No XP awarded separately. No HP. No stars.
- **Time budget:** 2–3 min (Standard) / 5 min (Extended). Extension beyond time budget is not allowed — Phase 6 must still fit in the session window.
- **AI Tier:** Tier 1 (pre-generated; no live AI required).

## Inputs

- `keywords_80_20` list extracted during Phase 2 pipeline (the ~20% of words carrying ~80% of meaning for the chapter).
- Content structure tag (hierarchical / sequential / spatial / discrete / numeric) — determined from the lesson archetype and subject framework.
- Subject family tag — narrows technique selection via subject-technique fit table above.
- Grade band — gates technique availability (Major System G7+ only; Memory Palace G3+).
- Archetype tag — determines whether Phase 5 fires at all (see trigger rule) and whether Standard or Extended mode applies.
- Session mode (Standard vs Extended) — set upstream by session orchestrator.

## Outputs

Phase 5 produces no scored data for the student leaderboard. It produces:

- `phase_5_complete: true` flag → required before Phase 6 can unlock.
- `technique_used` tag (memory_palace / peg_system / link_system / radiant_summary / major_system) → logged to session record for telemetry.
- `smashin_scope_score` per exercise → logged to content-pipeline QA record (not shown to student).
- `heritage_anchor` boolean per exercise → logged for 20%-injection compliance tracking.
- Attempt data (attempts per exercise, whether reveal was used) → passed to Phase 7 Reflection as part of per-phase data.

## Subject-specific examples

**Aniq Fanlar (Matematika — Grade 5, Fractions chapter):**
- Content structure: Discrete facts (5 fraction operation rules) → Peg System.
- Exercise 1: Student matches each operation rule (qo'shish, ayirish, ko'paytirish, bo'lish, tenglashtirish) to its Number-Shape peg (Sham, Oqqush, Yurak, Kema, Ilgak).
- Exercise 2: Same 5 rules presented out of order — student recalls the peg from memory.
- SMASHIN' SCOPE example: "Oqqush (Swan/2) suzayotib qo'lida ikki kasrni ushlab turadi — ularni bir maxrajga keltiradi" — vivid, humorous, movement-rich.

**Tabiy Fanlar (Biology — Grade 7, Cell Division chapter):**
- Content structure: Sequential (mitosis stages) → Link System.
- Exercise 1: 5-step chain (Interfaza → Profaza → Metafaza → Anafaza → Telofaza) assembled as a story: "Interfaza — hujayra ovqatlanadi; Profaza — xromosomalar ko'rinadi; ular Metafazada bitta qatorda tiziladi..."
- Exercise 2: Steps presented shuffled — student re-orders the chain from memory.
- Heritage anchor option: chain opens with *"Akademik Ibn Sino hujayrani kuzatdi..."* if organically fitting.

**Ijtimoiy Fanlar (Tarix — Grade 6, Silk Road chapter):**
- Content structure: Spatial (Silk Road city locations) → Memory Palace.
- Exercise 1: Registan Square as palace; student places 5 Silk Road cities (Samarqand, Buxoro, Xiva, Marg'ilon, Urganch) at the 5 hotspots.
- Exercise 2: Hotspots shown empty in shuffled label order — student re-places from memory.
- Heritage anchor: always active for Silk Road content (naturally 100% heritage).

**Til Fanlar (English — Grade 8, Conditionals chapter):**
- Content structure: Discrete facts (3 conditional types + usage rule) → Peg System.
- Exercise 1: Each conditional type (Zero, First, Second) paired to a peg image with a vivid sentence: "Candle (1) — Zero Conditional: 'If you heat water, it boils' — the candle always melts when lit."
- Exercise 2: Reverse recall — peg images shown, student types the conditional type name.

**Aniq Fanlar — G8 Algebra (Major System example):**
- Content structure: Numeric (year 1202 — Fibonacci sequence discovered by Leonardo of Pisa).
- Exercise: 1202 → 1=T, 2=N, 0=S (silent), 2=N → TNSN → encode as "Tosin" — student builds the sentence *"Tosin Fibonachchi sonlarini sanadi 1202 yilda."*

## Verification rules

1. **Trigger gate enforced:** If archetype is Concept Intro with a single concept, Phase 5 must not render. Test: check session payload — `phase_5_rendered` must be `false` when archetype=concept_intro and concept_count=1.
2. **Technique matches content structure:** The `buzan_technique` tag must match the `content_structure` tag per the mapping table. Test: validate `(content_structure, buzan_technique)` pair against allowed combinations.
3. **Major System grade gate:** If `buzan_technique = "major_system"`, grade band must be G7+. Test: assert grade_band ≥ 7 when technique is Major System.
4. **SMASHIN' SCOPE score ≥6:** Every exercise must carry `smashin_scope_score` ≥6 before deployment. Test: automated check on all Phase 5 content items in pipeline.
5. **No assessment elements:** Phase 5 must contain zero HP-counting or score-counting elements. Test: search session payload for `hp_delta`, `score_delta` events during Phase 5 — must return empty.
6. **No banned mechanics:** Phase 5 must not reference Tile Match, Sentence Fill, Tic Tac Toe, Connect Four, Mystery Box, Puzzle Lock, Word Ladder, Creative Lab, Adaptive Quiz. Test: keyword scan on phase mechanic tags.
7. **Failure language compliance:** Phase 5 student-facing messages must not contain *"Hali emas!"*, *"Noto'g'ri"*, or *"Xato"*. Test: string scan on all Phase 5 feedback strings.
8. **Pronoun compliance:** All Uzbek text must use "Siz". Zero "sen" occurrences. Test: regex scan.
9. **Time budget not exceeded:** Phase 5 must complete within 3 min (Standard) or 5 min (Extended). Session orchestrator must enforce hard cutoff. Test: assert phase_time_seconds ≤ 180 (Standard) or ≤ 300 (Extended).
10. **Completion flag emitted:** `phase_5_complete: true` must be in session state before Phase 6 renders. Test: assert flag presence in session context.

## UX/animation spec

**Layout:** Full-screen exercise card. Progress indicator: "Exercise 1 of 2" pill at top right. Phase name label: "Mustahkamlash" (Uzbek) / "Закрепление" (Russian).

**Entry card animation:** Fade-in from white (300ms). Motivational framing text appears with the same staggered sentence entrance used in Phase 0-A (opacity 0→1, translateY 8px→0, 300ms per block, 80ms stagger).

**Technique intro animation:** Compact modal overlay (bottom sheet, 60% screen height). Worked example rendered inline with color-highlighted keywords. Dismiss by tapping "Tushunarli!" (Got it!) — sheet slides down (350ms ease-in-out).

**Drag-and-drop (Radiant Summary, Memory Palace):**
- Draggable chips: rounded rectangles, subject-accent color border, drop shadow.
- Drop zones: dashed-border target areas. On correct drop: zone fills with subject-accent color + checkmark pulse (scale 1→1.2→1, 200ms). On incorrect drop: chip shakes (translateX ±8px, 3 cycles, 300ms) and returns to origin.
- Radiant Summary: central image fixed center; branches radiate outward at equal angles; drop targets at branch endpoints.

**Chain assembly (Link System):**
- 5 items displayed as stacked cards in scrambled order. Student drags to reorder vertically.
- Correct final order: each card snaps into position with a "click" micro-animation (scale 1.05→1, 150ms). Story chain text appears below the assembled sequence.

**Peg-to-concept pairing (Peg System):**
- Two-column layout: peg images (left), concept chips (right). Student drags concept chips to peg image slots.
- Correct pair: both items highlight in green and lock (no further dragging). Incorrect: shake and release.

**Word-digit conversion (Major System):**
- Digit sequence displayed as large numeral chips. Consonant mapping shown as a reference strip at bottom (always visible — student is learning the system, not tested on memorization of the code itself). Text input field for word submission.

**"Yana bir marta" retry button:** Ghost button below exercise. Always visible after first submission. Tapping resets drag state — no animation penalty.

**Transition to Phase 6:** Slide + blur (same as Phase 0-A panel transitions). Closing text: *"Yaxshi! Endi Boss uchun tayyormiz."* displayed for 2 seconds before auto-advance.

**Offline indicator:** If session is in offline mode, a small grey badge reads "Offline rejim" in the top-left corner. No functional change — Phase 5 runs identically offline.
