# Aniq Fanlar (Pure Mathematics)

## Purpose
Pure mathematics — arithmetic, algebra, geometry, analysis. Builds systematic problem-solving and abstract reasoning through numerical and spatial concepts.

## Subjects

| Subject | Uzbek Name | Grades |
|---------|------------|--------|
| Mathematics | Matematika | G1–6 |
| Algebra | Algebra | G7–9 |
| Geometry | Geometriya | G7–9 |
| Algebra & Analysis | Algebra va Matematik Analiz | G10–11 |
| Geometry (Advanced) | Geometriya (Advanced) | G10–11 |

## Learning outcomes
- Recognize mathematical structures and apply appropriate solution methods
- Translate real-world problems into mathematical equations (and reverse: formula → meaning)
- Execute multi-step procedures with fully shown work
- Evaluate whether solutions make physical or logical sense
- Build spatial reasoning through geometric visualization

## Teaching methodology
Universal methods from `00-core/core-principles.md §1.4`: two-layer explanation, don't-assume.

Family-specific (TM-1 → TM-4 — shared with Tabiy Fanlar):
- **TM-1 Genetic method:** Teach by retracing the concept's historical birth — original problem → struggle → invention → formalization → today's shorthand. The formula becomes inevitable, not arbitrary.
- **TM-2 No magic moves:** Every algebraic manipulation traced to its source. If a step would make a student ask "where did that come from?", the content answers BEFORE they ask.
- **TM-3 Origin journey IS the teaching:** Preview Panel 4 (Origin) is the narrative backbone — not history trivia.
- **TM-4 Derivation transparency:** Expand every template before using it. No step is assumed valid.

## Easy mode pipeline
```
Preview (Panels 1,2,3,4) → Flash Cards (5-8) → Sprint (5 items) → Games (2: Adaptive Quiz + Tile Match OR Memory Match) → Reflection
```
- Panel 4 (Origin) included — genetic method applies
- Adaptive Quiz mandatory; Notebook Capture on all calculation steps inside it
- ~35–40 min

## Hard mode pipeline
```
Preview (Panels 1-6) → Flash Cards (8-12) → Sprint (7 items) → Games (3: Adaptive Quiz + Tile Match + Memory Match Blitz) → Real-Life (1 deep scenario) → [Consolidation if ≥2 concepts] → Final Challenge (Boss) → Reflection
```
- Real-Life: single immersive scenario, modern professional Uzbek context (IT Park, solar, UAV, Metro — see `00-core/context-policy.md`)
- Capture on all calculation steps across all phases
- Final Challenge: Standard Boss for Algorithm/Application; Full Boss for Review
- ~70–90 min

## Key rules (compact)

- **AF-1:** Adaptive Quiz mandatory in Phase 3, both modes (`01-phases/phase-03-game-breaks.md`)
- **AF-2:** Notebook Capture on any multi-step calculation (`00-core/capture-rule.md`); G5-8: ≥1/3 sessions; G9-11: ≥1/2 sessions
- **AF-3:** No MC for G6+ in Final Challenge; G5: ≤30% MC; G1-4: MC allowed (`00-core/hp-damage-baseline.md`)
- **AF-4:** Matematika G5-6 HP override = 80 (not 100); all other Aniq Fanlar G5-8 = 100
- **AF-5:** Bidirectional thinking required every session — ≥1 real-world→formula AND ≥1 formula→real-world item
- **AF-6:** Error Analysis/Detection primary mechanic — appears ≥1 game every 2 sessions (per-subject tracking)
- **AF-7:** All numerical answers must include units or context labels; `x = 5` scores as incomplete
- **AF-8:** CPA progression mandatory — every new concept: Concrete → Pictorial → Abstract; no abstract-only intro
- **AF-9:** Pronoun policy — "Siz" (formal) always, never "sen" (`00-core/pronoun-policy.md`)
- **AF-10:** Removed/frozen games — never select: Blackjack 21, Bridge Builder, Minefield Navigator, Escape Room, Codebreaker. v2+ frozen: Why Chain, Territory Conquest, Connect Four, Word Ladder, Puzzle Lock, Reaction Chain.
