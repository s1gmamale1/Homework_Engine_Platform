# Math + Algebra — Hard Mode

## Purpose

Matematika (G5-6) and Algebra (G7-9) are the same progression: numbers → patterns → symbols → equations. Matematika builds arithmetic fluency and proportional reasoning with concrete objects. Algebra generalizes that into symbolic language — variables, expressions, equations, functions.

**Core demand:** Translate between real-world situations and mathematical representations. Every lesson should strengthen the bridge: word problem ↔ formula ↔ visual model.

**Grade bands:**

| Grade | Stage | Key shift |
|-------|-------|-----------|
| G5-6 | Concrete → Pictorial → Abstract (CPA mandatory every session) | Fractions, decimals, percentages, area, basic ratios |
| G7 | Variables need context anchors ("let x = number of cases sold") | Linear equations, one variable |
| G8 | Equation setup from word problems | Two-variable, systems |
| G9 | Full symbolic abstraction | Quadratic, non-linear intro |

**Teaching rules:**
- **CPA:** G5-6 must show concrete object → diagram → formula for every new concept. G7-9 compresses concrete phase but never skips pictorial.
- **No magic moves:** Every algebraic step traced to its source. If a student would ask "where did that come from?", answer BEFORE they ask.
- **Genetic method:** Teach by retracing the concept's origin — original problem → struggle → invention → formalization. The formula becomes inevitable, not arbitrary.
- **Derivation transparency:** Expand every formula template before using it. No step assumed valid.
- **Units always required:** `x = 5` is incomplete. `x = 5 m` or `x = 5 ta` is correct.
- **Bidirectional thinking:** Every session has ≥1 real-world→formula AND ≥1 formula→real-world item.
- **Error Analysis:** Appears ≥1 game every 2 sessions — find and fix mistakes in worked solutions.

---

## Hard Pipeline (7-8 components)

```
Preview → Flash Cards → Memory Sprint → Game Breaks (3 games) → Real-Life → [Consolidation if ≥2 concepts] → Final Challenge → Reflection
```

Practical time: ~50-53 min (cap: 60 min). Learning time (Preview + Flash Cards): ~23 min.

---

### 1. Preview (7 panels)

**Time:** ~15 min | **Not scored**

7 panels. Full depth. Panel 4 (Origin) = genetic method. Panel 5 (Word→Formula) = bridge to Real-Life.

| Panel | What to build |
|-------|---------------|
| **1. Summary** | Reframe the chapter as a PROBLEM, not a definition. "Nima uchun bu kerak?" Visual overview — concept map, equation tree, or chapter roadmap. Structural scan with bold-term chips. |
| **2. Better Explanation** | CPA steps: concrete analogy → labeled diagram → notation/formula. Explain what the textbook SHOULD have said. Each sentence adds understanding — no filler. |
| **3. Examples** | 3-5 worked examples, simple → complex. Each: setup → visual → step-by-step → answer with units. |
| **4. Origin** | Genetic method: who invented this, what problem forced its creation, how it evolved. Tell the truth — if Pythagoras invented it, say Pythagoras. If Al-Xorazmiy, say Al-Xorazmiy. National pride comes from honesty, not forced attribution. Write as a discovery STORY — this IS teaching, not trivia. |
| **5. Word → Formula Translation** | THIS IS THE BRIDGE TO REAL-LIFE. Teach the student HOW to: (1) Read a real-world problem and understand what's being asked, (2) Extract the relevant data — what numbers matter, what's noise, (3) Convert extracted data into this chapter's formula/equation. Show 2-3 mini-cases: real sentence → underline key data → write the formula → solve. Start simple, build to chapter-level complexity. Without this panel, Real-Life Challenge becomes guesswork. |
| **6. Industry Application** | Where professionals use this math. Pick 2-3 roles: muhandis, tahlilchi, dasturchi, me'mor, tadbirkor. Show SPECIFIC tasks with consequences: "Me'mor xona rejasini chizayotganda yuza formulasidan foydalanadi — formula noto'g'ri bo'lsa, bino sig'maydi." Not vague — concrete task + what breaks if wrong. |
| **7. Why This Matters** | What breaks if you DON'T know this + what opens UP if you DO. End with BOST goal: "Bugun [topic] haqida nimani bilmoqchisiz?" — stored for Reflection. |

**Two-layer explanation in every panel:**
1. Full mathematical rigor
2. "Sodda so'zlar bilan" — everyday Uzbek, no jargon

**Language:** "Siz" throughout. Average sentence ≤14 words (G5-6) or ≤16 words (G7-9).

### 2. Flash Cards (8-12 cards)

**Time:** ~8 min | **Not scored**

- Priority: Formula → Rule → Definition
- Every card: formula WITH visual example — no symbol-only cards
- Include 1 "Common Mistake" card — show error + fix
- G5-6: 8 cards. G7-9: 10-12 cards.
- G7-9: include notation cards (balance notation, function notation)
- Cards returnable throughout homework

---

### 3. Memory Sprint (7 items)

**Time:** ~3 min | **Weight: 10%**

Warm-up. Tap-only. MC + T/F + YNNG. Must mix ≥2 formats.

**Distribution:** 3 MC + 2 T/F + 2 YNNG (7 items)

**Construction:**

**MC (4 options):**
- Distractors map to common errors: wrong operation, sign/place-value error, partial answer
- G7-9: "equivalent expression" items, error-spot items ("Bu yechimda xato qayerda?")

**T/F:**
- Test rules or misconceptions, never trivial arithmetic
- G7-9: algebraic identities — "x² + 2x + 1 = (x+1)² — To'g'ri yoki Noto'g'ri?"

**YNNG:**
- Chapter comprehension. ≥1 "Aytilmagan" answer.
- G7-9: "Bobda aytilganiga ko'ra, har qanday kvadrat tenglama ikkita yechimga ega — Ha / Yo'q / Aytilmagan"

**G5-6:** Include 2-3 mental math speed rounds (3-sec timer).

---

### 4. Game Breaks (3 games)

**Time:** ~20 min | **Weight: 30%**

**Mandatory:** Adaptive Quiz in one slot (Notebook Capture on every solve step).
**Remaining 2 slots:** Pick from v1 pool based on chapter content. Slot 2 is the Von Restorff Anchor — load outstanding/surprising items there.

**v1 game pool (all 5 available):**

| Game | Math/Algebra use | When to pick |
|------|-----------------|--------------|
| **Adaptive Quiz** | Multi-step calculation, equation solving | MANDATORY — always 1 slot |
| **Tile Match** | Formula ↔ visual, expression ↔ simplified, equation ↔ graph | Concept has visual/symbolic pairs to match |
| **Memory Match** | Formula �� name, term ↔ definition, identity pairs | Chapter has 8+ terms/formulas to pair |
| **Sentence Fill** | Drag equation parts into position, fill missing step/operation, cloze formula | Procedural chapters — solving steps, order of operations, proofs |

**Construction notes:**
- **Tile Match:** 6-8 pairs. G5-6: fraction ↔ visual, formula ↔ bar model. G7-9: expression ↔ equivalent, equation step ↔ justification. Von Restorff slot: load surprising math facts, unusual visuals.
- **Memory Match:** 4×4 grid (8 pairs). G5-6: number ↔ visual. G7-9: identity pairs, function ↔ graph.
- **Sentence Fill:** 5-8 items. Gap = the operation, variable, or next step. Must test mathematical understanding, not random word removal. G7-9: fill missing algebraic manipulation + justification.
- **Notebook Capture (standalone):** 3-5 problems, camera verify each. Use when chapter is pure calculation (long division, multi-step equations).

**Capture:** Notebook Capture on Adaptive Quiz only (AF-1). Other games = no capture unless Notebook Capture selected as standalone.

### 5. Real-Life Challenge (1 deep scenario)

**Time:** ~15 min | **Weight: 30%**

ONE scenario, 4-6 sub-questions. Student is THE EXPERT — first-person POV mandatory.

**Expert roles:**
- G5-6: Project coordinator (materials budget), logistics planner (delivery routes), data analyst (sales charts)
- G7-9: Business analyst (break-even), network engineer (comparing plans), urban planner (budget constraints), sports analyst (performance trends)

**Construction:**
1. **Context narrative** (3-4 sentences): modern professional Uzbek setting. "Siz [role]. Sizning vazifangiz..."
2. **Sub-questions** (4-6): each tests a skill taught in Preview. Multi-step calculation required.
3. **Scope lock:** Tests APPLICATION of Preview content only. No new concepts. Same difficulty tier as hardest Preview example, different numbers/context.

**Scaffolds:**
- **W5H:** G5-6 mandatory (4/6 branches). G7-8 default ON. G9 toggleable ("Tahlil" button). Pure calculation exempt.
- **Decision Map:** Only for genuine two-option trade-offs (comparing plans, choosing methods). Student maps pro/con, assigns weights, justifies.

**Capture:** Notebook Capture on every calculation solve-step. 1-3 capture points per scenario.

**Wise Status (30% of sessions):**
- Professional title: "Bosh Muhandis," "Loyiha Menejeri"
- Arena 55/45: national (Yashil Makon, IT Park, Afrosiyob) / global
- Academic core UNTOUCHED — framing only
- Closing: "Sizning aniqligingiz Uchinchi Renessansni quradi"

**Context policy:** No bazaar/village/shopkeeper/farmer. Modern professional only. IT Park, solar farms, metro, startups, Yashil Makon.

**Variable rule (G7-9):** Student must define variable explicitly and set up equation from scratch — not plug into given formula.

### 6. Consolidation (conditional — only if ≥2 concepts)

**Time:** ~3 min | **Not scored**

Fires ONLY when the lesson teaches ≥2 distinct interlocking concepts. Single-concept lessons skip this.

**Technique by content structure:**

| Structure | Technique | Example |
|-----------|-----------|---------|
| Hierarchical (fraction types, equation taxonomy) | Radiant Summary | Center: "Kasrlar" → branches: To'g'ri / Noto'g'ri / Aralash |
| Discrete formulas (5 key rules) | Peg System | Number-shape pegs, each formula paired with image |
| Sequential (procedure steps) | Link System | Steps of long division or completing the square in a vivid story chain |
| Spatial (area formulas, grid geometry) | Memory Palace | Registan Square — each shape at a hotspot with formula |

Followed by: formula-visual pairing check (45 sec) — show formula, student selects correct visual.

---

### 7. Final Challenge (Boss)

**Time:** ~10 min | **Weight: 30%**

HP combat. Damage tiers: Easy -10, Medium -20, Hard -30. Distribution: 40/40/20.

| Grade | HP | MC allowed? |
|-------|:--:|:-----------:|
| G5-6 | **80** (Matematika override) | ≤30% MC |
| G7-8 | 100 | **No MC** |
| G9 | 100 | **No MC** |

**Boss question construction:**
- ≥1 word problem (text → math translation)
- ≥1 visual model question (bar model, grid, graph, diagram)
- G7-9: ≥1 modeling question (student writes equation from scenario) + ≥1 interpretation ("What does your answer mean?")
- No purely symbolic questions without context
- No question requiring >3 calculation steps without scaffolding (G5-6)
- Every question tagged: `[Bloom: LX | PISA: LX | Damage: -XX HP]`

**Hint ladder:** -5 HP per hint (G5-6 gentler). Socratic style — show first step, ask student to complete.

**Failure:** "Hali emas!" — never "Noto'g'ri". Route to relevant Preview panel, regenerate with same standards, different numbers.

---

### 8. Reflection

**Time:** ~2 min | **Not scored (analytics)**

1. **Summary (3-5 sentences):** Core concept + key formula as visual card + real-world connection from Real-Life.
2. **3 metacognitive prompts** (student picks 1):
   - "Bu sessiyada eng qiyin [tenglama/masala] qaysi edi? Nima uchun?"
   - "Agar sinfdo'stingizga tushuntirsangiz, birinchi qanday boshlardingiz?"
   - "Bu mavzuni kundalik hayotda qayerda ishlatishingiz mumkin?"
3. **BOST Goal Recall:** Resurface stored goal from Preview Panel 6. "Bildingizmi?" → Ha / Qisman / Hali yo'q.
4. **Spaced repetition:** "Keyingi takrorlash: ertaga, 3-kun, 7-kun, 21-kun." Flag 2-3 most-missed concepts.
5. **Closing:** ≥60% → "Sizning aniqligingiz va qat'iyatingiz — Uchinchi Renessansning poydevori." <60% → "Har bir mashq miyangizni kuchaytirad. Ertaga davom etamiz!"

## Rules (compact)

- **MA-H1:** Adaptive Quiz mandatory in Game Breaks, only game with Notebook Capture
- **MA-H2:** CPA mandatory G5-6. G7-9 compresses concrete but never skips pictorial.
- **MA-H3:** No MC in Final Boss for G6+. G5: ≤30%.
- **MA-H4:** Matematika G5-6 HP = 80. All other = 100.
- **MA-H5:** Bidirectional thinking every session: ≥1 real→formula + ≥1 formula→real
- **MA-H6:** Error Analysis every 2 sessions minimum
- **MA-H7:** All answers must include units or context labels
- **MA-H8:** Real-Life scope locked to Preview content — no new concepts
- **MA-H9:** Pronoun: "Siz" always, never "sen"
- **MA-H10:** No bazaar/village cliches — modern professional contexts only
- **MA-H11:** v1 games: Adaptive Quiz (mandatory, Notebook Capture built in), Tile Match, Memory Match, Sentence Fill. Pick based on chapter content — don't default to same combo every session.
