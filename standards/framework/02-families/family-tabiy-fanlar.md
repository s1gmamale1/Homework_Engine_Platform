# Tabiy Fanlar (Natural Sciences)

## Purpose
Natural sciences — physics, chemistry, biology. Understanding natural phenomena through observation, experimentation, and mathematical modeling.

## Subjects

| Subject | Uzbek Name | Grades |
|---------|------------|--------|
| Physics | Fizika | G7–11 |
| Chemistry | Kimyo | G8–11 |
| Biology | Biologiya | G5–11 |

## Learning outcomes
- Understand natural processes and their underlying mechanisms
- Apply mathematical models to physical, chemical, and biological systems
- Conduct experimental reasoning: hypothesis → observation → conclusion
- Connect microscopic causes to macroscopic effects
- Read and interpret scientific diagrams, data tables, and graphs

## Teaching methodology
Universal methods from `00-core/core-principles.md §1.4`: two-layer explanation, don't-assume.

Family-specific (TM-1 → TM-4 — same as Aniq Fanlar):
- **TM-1 Genetic method:** Trace concepts to their historical or experimental origin before formalizing
- **TM-2 No magic moves:** Every derivation or mechanism step justified
- **TM-3 Origin journey IS the teaching:** Panel 4 (Origin) is backbone, not sidebar
- **TM-4 Derivation transparency:** Expand formulas before applying them

Additional — **Photosynthesis pattern (Real-Life):** Tell the process as a compact flowing story first (matching textbook exactly — no invented steps), then ask 2–3 questions from it. This replaces the scenario-problem format used in Aniq Fanlar.

## Easy mode pipeline
```
Preview (Panels 1,2,3,4) → Flash Cards (5-8) → Sprint (5 items) → Games (2: Adaptive Quiz OR Tile Match + Memory Match) → Reflection
```
- Panel 4 (Origin) included — genetic method applies to sciences
- Capture on Physics/Chemistry calculation steps; Biology: no capture
- ~35–40 min

## Hard mode pipeline
```
Preview (Panels 1-6) → Flash Cards (8-12) → Sprint (7 items) → Games (3: Adaptive Quiz OR Tile Match + Memory Match + Tile Match OR Sentence Fill) → Real-Life (1 narrative-process scenario) → [Consolidation if ≥2 concepts] → Final Challenge (Boss) → Reflection
```
- Real-Life: photosynthesis pattern — flowing process story → questions (not word-problem format)
- Tile Match / Memory Match for sequential processes (metabolic pathways, reaction sequences, physical process chains) — Reaction Chain is frozen to v2+
- Capture: Physics/Chemistry calculation steps only; Biology exempt
- ~70–90 min

## Key rules (compact)

- **TF-1:** Notebook Capture on calculation steps — Physics: formula substitution steps; Chemistry: stoichiometry/mole steps; Biology: exempt (`00-core/capture-rule.md`)
- **TF-2:** Adaptive Quiz mandatory status — open question. Working assumption: mandatory for Physics + Chemistry (parallel to Aniq Fanlar AF-1); Biology: optional. Resolve before first Physics/Chemistry build.
- **TF-3:** Tile Match or Memory Match for process-sequence content (Reaction Chain is frozen to v2+)
- **TF-4:** Real-Life uses narrative-process format (photosynthesis pattern), not scenario-problem format
- **TF-5:** Phase 2 (Reading) is always skipped — not a language family
- **TF-6:** Consolidation fires only when lesson covers ≥2 distinct interlocking concepts; skipped for single-concept lessons even in Hard mode
- **TF-7:** Pronoun policy — "Siz" (formal) always, never "sen" (`00-core/pronoun-policy.md`)
- **TF-8:** Modern professional contexts only — lab, pharma, agritech IoT, climate analysis, medical diagnostics (`00-core/context-policy.md`)
- **TF-9:** Removed/frozen games — never select: Blackjack 21, Bridge Builder, Minefield Navigator, Escape Room, Codebreaker. v2+ frozen: Why Chain, Territory Conquest, Connect Four, Word Ladder, Puzzle Lock, Reaction Chain.
