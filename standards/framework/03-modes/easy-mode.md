# Easy Mode — 5-Component Homework

## Definition

A lighter homework session for introductory lessons, vocabulary-heavy content, and early-chapter sections. Approximately 35-40 minutes total.

---

## When to use

- First section of a new topic (new term/object introduced)
- Vocabulary or terminology lessons
- Early-chapter sections with light content
- G5-6 default for most subjects unless teacher overrides to Hard

---

## Detection rule

Content introduces a new term/definition that doesn't appear in any prior section AND involves no systematic procedure. Teacher can tag manually at pipeline entry.

---

## Active components

EXACTLY 5 components, same structure for ALL subject families.

| # | Component | Phase file | Scoring weight | Time |
|---|-----------|------------|:-:|---|
| 1 | Preview | phase-0a-preview.md | — (not scored) | ~10 min |
| 2 | Flash Cards | phase-0b-flashcards.md | — (not scored) | ~5 min |
| 3 | Memory Sprint | phase-01-memory-sprint.md | 20% | ~3 min |
| 4 | Game Breaks | phase-03-game-breaks.md | 50% | ~15 min |
| 5 | Reflection | phase-07-reflection.md | — (analytics) | ~2 min |
| | **Total** | | **70% graded** | **~35 min** |

> Note: scoring weights (20/50) are proposals, pending pilot calibration. Reflection adds 30% for routing purposes as a combined "session score."

---

## Skipped components

Reading (Phase 2), Real-Life (Phase 4), Consolidation (Phase 5), Final Challenge (Phase 6).

---

## Preview panel selection (Easy mode)

Easy uses 3-4 panels from the 8 possible. Mandatory core + optional per family:

| Panel | Easy mode | Why |
|-------|:---:|---|
| 1. Summary | ✅ always | What this is about |
| 2. Better Explanation | ✅ always | Clarify the confusing parts |
| 3. Examples | ✅ always | Show the concept in action |
| 4. Origin / Real-Life Research | ✅ Math/Science only | Genetic method needs it (TM-1) |
| 5. Personal Hook | ❌ | Hard mode extra |
| 6. Why This Matters | ◐ optional 4th/5th | If room and motivation needed |
| 7. Industry Application | ❌ | Hard mode extra |
| 8. Additional Materials | ❌ | Hard mode extra |

**Easy minimum = Panels 1, 2, 3.** Math/Science add Panel 4 (origin journey). Panel 6 optional.

## Component scaling

| Component | Easy | Hard |
|-----------|------|------|
| Preview panels | 3-4 | 5-8 |
| Flash Cards | 5-8 cards | 8-12 cards |
| Memory Sprint | 5 items | 7 items |
| Game Breaks | 2 games | 3 games (2 default + 1 interactive) |

---

## Pass threshold

60% of graded components. Below 60% → remediation: re-show relevant Preview panels + Flash Cards + easier variants of failed items.
