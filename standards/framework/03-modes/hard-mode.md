# Hard Mode — 6-8 Component Homework

## Definition

Full-depth homework for method/formula lessons, application problems, chapter reviews, and summative assessments. Approximately 70-90 minutes total.

---

## When to use

- Systematic procedure taught (formula, algorithm, method)
- Application / word-problem sections
- End-of-chapter review
- G9-11 default for STEM
- Any lesson where teacher or pipeline judges "student needs deep practice"

---

## Detection rule

Section teaches a systematic procedure (grep for "formula", "algorithm", "method", "usul", "qoida") OR references ≥2 prior sections. Teacher can tag manually at pipeline entry.

---

## Component scaling in Hard mode

| Component | Hard mode |
|-----------|-----------|
| Preview | 5-8 panels (full depth) |
| Flash Cards | 8-12 cards |
| Memory Sprint | 7 items |
| Game Breaks | 3 games (2 default + 1 interactive); Adaptive Quiz mandatory + capture for Math/Science |
| Real-Life | ONE deep immersive scenario with capture on solve steps |
| Final Challenge | Standard or Full Boss (HP combat active) |

---

## Pass threshold

60% of graded components. Below 60% triggers full remediation path.
