1. What Ona Tili Teaches

Ona tili in NETS is a native language subject. Students already speak Uzbek fluently. Homework formalizes and sharpens what they already know intuitively.

  • Grammar awareness: Focus on analysis, structure, punctuation, and formal register.
  • Transformation: Transform dry textbook grammar drills into engaging homework.
  • Story Mode: Make flat textbook explanations vivid.
  • Active Analysis: Turn boring drills into Game Breaks and Real-Life Challenges.

Ona Tili is NOT about: Basic communication, word-translation, or heavy vocab acquisition.

3. Content Production Rules by Grade Band

G5-6 (Foundation)

  • Grammar through sentence context, never isolated rule recitation first.
  • Focus on parts of speech, spelling rules (apostrophes), and basic punctuation.
  • Use chunking: break complex sentences into parts, then reassemble.
Good Pattern: "Aziza wrote a letter but forgot all the commas. Help her fix it and explain why each comma matters."

G7-9 (Development)

  • Emphasize complex sentence breakdown and subordinate clauses.
  • Use morphology actively: root -> suffix -> meaning chain.
  • Teach word formation as a system (how suffixes shift meaning).
Good Pattern: "You found two versions of the same message: one informal, one formal. Compare them and rewrite the informal version for a school newspaper."

G10-11 (Advanced)

  • Require full morphological decomposition.
  • Add stylistic analysis of literary and formal texts.
  • Formal composition must include thesis, evidence, and conclusion.
Good Pattern: "You are revising a formal article for publication. Identify weak sentence structure, improve punctuation, and explain how the revision changes tone."
4. Game Mechanics — Ona Tili Specific

Primary Mechanics

  • Sentence Fill / Cloze: Core mechanic, test grammar in context.
  • Tile Match: Use for sentence structure, not vocab matching.
  • Story Mode: Narrative wrap for grammar/editing/punctuation.
  • Why Chain: Grammar reasoning and structure awareness (up to 3 levels).

Interactive Picks

Game Best Use
Codebreaker Word decomposition (root + suffix)
Reaction Chain Sentence parsing sequence
Puzzle Lock Paragraph or text reconstruction
Memory Match Blitz Grammar rule <-> Example pairs

Phase 5 Buzan Options

  • Radiant Summary: Rule hierarchies, suffix trees.
  • Link System: Morphological chains as memorable sequences.
  • Peg System: Punctuation rules anchored to stable pegs.

Games to Avoid

Word Ladder, Listening Tasks (use Dictation instead), Notebook Capture.

5. Boss Format

Combine writing production + grammar analysis.

  • G5-6: Passage correction + short production (2-3 sentences).
  • G7-8: Structure parse + rewrite in different register.
  • G9-11: Analytical writing with explicit grammar/style focus.
  • Failure framing: Use TEFCAS language ("Hali emas!").
Good Boss (G7): "Break this sentence into clauses. Label each: main/subordinate. Rewrite it as two simple sentences."
6. Story Mode & Real-Life Challenge

Story Mode

Turn grammar into a narrative problem: editing, proofreading, rewriting, or structure repair.

Good Example: "You received a letter full of errors. Each segment has 3 mistakes. Find and explain each."

Real-Life Challenge

The student becomes the language expert.

  • G5-6 Helper: Fix a class notice with punctuation mistakes.
  • G7-8 Editor: Edit a junior reporter's article with grammar errors.
  • G9-10 Official: Rewrite informal message as official school announcement.
  • G10-11 Critic: Write an official speech for Independence Day ceremony.
