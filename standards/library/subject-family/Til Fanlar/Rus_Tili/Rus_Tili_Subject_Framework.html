1. What Rus Tili Teaches

Russian in Uzbekistan is unique, acting as either a second language (L2) or a primary academic language (L1).

  • UZ-stream (L2): Focus on scaffolding, vocabulary, and grammar acquisition.
  • RU-stream (L1): Focus on grammar analysis, formal writing, and composition (similar to Ona tili).
  • Stream Detection: Inspect the {lang} field to determine the production mode.
  • Transformation: Bridge the gap between dense textbook material and student proficiency using Story Mode and Game Breaks.

Rule: Determine the stream first. The wrapper must match whether the student uses Russian as L1 or L2.

3. Content Production Rules by Grade Band

G5-6 (Foundation)

  • UZ-Stream: 8-10 new vocab items.Pattern-based grammar. 50-100 word passages.
  • RU-Stream: Native-language logic. No scaffolding for everyday words. 80-120 word passages.
Good RU Pattern: "The school notice has mistakes. Fix them and explain the rule."

G7-9 (Development)

  • UZ-Stream: Reduce translation dependence. Functional grammar in social/school scenarios.
  • RU-Stream: Deep syntax and morphological analysis. Rewriting tasks (informal -> formal).
Good UZ Pattern: "Help a Russian tourist in Samarkand. Identify the need and write a reply."

G10-11 (Advanced)

  • UZ-Stream: Formal composition and density. Extended response with clear argument.
  • RU-Stream: Full rhetorical analysis. Extended analytical writing. Professional roles (critic, speechwriter).
4. Game Mechanics — Rus Tili Specific

Sent. Fill Contextual grammar Core mechanic (ID/Fix)
Listening Comprehension speed Not primary (use Dictation)
Story Mode Functional scenarios Editing/Narrative wrap

Interactive Picks

  • Codebreaker: Perfect for Russian morphology (root/prefix/suffix).
  • Reaction Chain: Sentence parsing or dialogue flow.
  • Puzzle Lock: Paragraph reconstruction / coherence.
  • Memory Match Blitz: UZ: Translation pairs; RU: Rule-Example pairs.

Note: Skip Notebook Capture and Speed Match (math-focused).

5. Boss Format
  • UZ-Stream: Scaffolded production (Guided -> Prompted -> Free).
  • RU-Stream: Grammar analysis + Composition (Native level).
  • HP: G5-8 = 100, G9-11 = 150.
  • Failure framing: Use TEFCAS (UZ: "Hali emas!", RU: "Eshyo ne vsyo!").
Good RU G7-9: "Rewrite the passage in formal register and explain two grammar choices."
6. Story Mode & Real-Life Challenge

Story Mode

  • UZ G7: Help a tourist navigate Samarkand and communicate with locals.
  • RU G8: Act as editor for the school Russian newspaper; fix errors before print.

Real-Life Challenge

Must include Role, Audience, and Purpose.

UZ G7: "Helping a lost tourist. Write the directions in Russian."
RU G8: "School newspaper editor. Fix 5 errors and explain corrections."
