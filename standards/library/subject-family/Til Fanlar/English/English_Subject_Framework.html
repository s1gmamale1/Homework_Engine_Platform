1. What English Teaches

English in NETS is a foreign language, not a native or heritage language. This changes the production rules:

  • i+1 calibration: Pacing is tighter to prevent session breakage.
  • CEFR Spine: A1 (G5) -> A2 (G6-7) -> B1 (G8-9) -> B2 (G10-11).
  • Transformation Philosophy: Transform textbook topics into strong homework; do not copy weak pedagogy. Use Story Mode for dry explanations and Real-Life Challenge for engagement.
  • Production lag: Production lags comprehension by 1-2 levels.

Rule: Assume less background knowledge, more support, and slower release from scaffolding.

3. Content Production Rules by Grade Band

G5-6 (Foundation)

  • Max 5 new vocab items per session.
  • Pair every word: image + UZ translation + audio + example.
  • Reading: 50-80 words max. Listening: 30 sec max.
  • Make it alive: Use Story Mode to add narrative and Real-Life Challenge for concrete purpose.
Good Phase 4: Tour guide role-play using SAMARKAND, BEAUTIFUL, OLD.

G7-9 (Development)

  • 8-12 new vocab items.
  • Reading: 100-200 words. Listening: 45-60 sec (natural speed).
  • Make it alive: Build narrative around target grammar; create role-based communication tasks.
Good Phase 4: Hotel receptionist responding to guest complaint.

G10-11 (Advanced)

  • 15+ vocab items (Academic sets).
  • Reading: 300-500 words. Listening: 2-3 min (authentic).
  • Make it alive: Story Mode should create interpretive tension; Real-Life Challenge should create professional or civic context.
Good Phase 4: Environmental consultant report (150-200 words).

  • Reaction Chain: Sentence/Dialogue flow (G7+).
  • Puzzle Lock: Paragraph reconstruction (G9+).

Tile Match Note: Use for word order, not formula matching.

5. Boss Format — English Specific
  • G5: Guided production (stems).
  • G7: Prompted (5 sentences, grammar req).
  • G11: Extended composition (150-200 words).
  • HP: G5-8 = 100, G9-11 = 150.
  • Failure: Use "Hali emas! Let's try again."
Bad Boss: "What is the definition of the present perfect?"
