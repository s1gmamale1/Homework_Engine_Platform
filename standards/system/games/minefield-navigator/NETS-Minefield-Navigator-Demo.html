Hazard Grid

Only adjacent moves are allowed. Start is top-left. Exit is bottom-right. The grid now uses simple Buzan-style spatial anchors: each zone represents a concept family so location supports memory, not just movement.

S = Start E = Exit Visited tiles stay visible Mines stay hidden until triggered Blue = Coordinates Green = Direction Amber = Route logic

Choose an adjacent tile

When you click a valid tile, one question appears. Correct answer keeps your chosen move. Wrong answer triggers a forced adjacent move. Use the zone anchors to remember what kind of question each region tends to represent.

function zoneFor(r, c) { if (c <= 1) return "zone-a"; if (c === 2) return "zone-b"; return "zone-c"; } function zoneLabel(r, c) { if (c <= 1) return "Coordinates"; if (c === 2) return "Direction"; return "Route Logic"; }
