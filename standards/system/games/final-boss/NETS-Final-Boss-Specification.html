The 3 Boss Types

DAILY Sub Boss
Every session. Current lesson. Hints available.
Basic: 2 attempts max. Premium: unlimited.
WEEKLY Big Boss
Cross-subject (weakest 3 standards). One PISA tier harder.
1 attempt default. Extendable via bonuses (TBD). 2x XP.
ELITE Mythical Boss
Random / easter egg / bonus credits (Basic) / Mystery Box (Premium).
PISA L5-6. Zero hints. One attempt. 5x XP + exclusive rewards.

TierAccess

Basic Sub Boss Failure Flow

Attempt 1 → FAIL (HP > 0)
  → Socratic tutoring (Tier 2). Questions regenerated.
Attempt 2 → FAIL (HP > 0)
  → Homework marked FAILED.
  → Duolingo Mode: task-by-task review, -1 difficulty tier, repeat until 60%.
  → Teacher flagged after 3 failed sessions on same topic.
  → No more boss attempts this session.

Premium Sub Boss Failure Flow

Attempt N → FAIL (HP > 0)
  → Socratic tutoring (Tier 3 — personalized, textbook-referenced).
  → Questions regenerated. Student re-attempts.
  → Unlimited until defeated.
  → Duolingo Mode still triggers if session < 60%.

Hints (Sub & Big only)

3 hints per question: vague → specific → directed. Each costs HP.
Basic: Hints from pre-written socratic_hints arrays.
Premium: Tier 3 AI — dynamically generated based on student's specific error.
Mythical: ZERO hints. Both tiers. No scaffolding.

Anti-Cheat

1. Question Regeneration — fresh every attempt
2. Response Time — faster than reading pace = flag
3. Pattern Analysis — perfect Hard + fail Easy = flag
4. Socratic Verification — 1 post-defeat follow-up (not scored, flagged if failed)
5. Copy-Paste Detection — paste events flagged
6. Radiance Score — 0.0-1.0 authenticity check
Escalation: soft warn → parent → teacher → admin

Edge Cases

Catch-Up (>7d absent): HP 50%, hints free (both tiers). Basic still 2 attempts.
Recovery: Boss unchanged — full integrity.
AI down: Tier 1 fallback. Session never terminates.
Mythical + Recovery: Suppressed (both tiers).
Teacher HP modifier: 50%-150%. Custom boss questions allowed.
