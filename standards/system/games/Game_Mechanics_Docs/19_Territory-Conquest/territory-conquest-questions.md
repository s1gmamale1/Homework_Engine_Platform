# Territory Conquest: Question Mechanics & Difficulty Mapping

This document defines the interactive styles used during Battle Challenges in Territory Conquest.

| PISA Level | Cognitive Target | Mechanic Style | Interaction Logic |
| :--- | :--- | :--- | :--- |
| **L1** | **Recall** | **Multiple Choice (MCQ)** | 4 Options, 1 Correct. Fast-paced retrieval. |
| **L2** | **Understand** | **Multi-Select** | Select all that apply (2-3 correct out of 5). |
| **L3** | **Apply** | **Tile Matching** | Match terms to definitions or causes to effects. |
| **L4** | **Analyze** | **Connect the Dots** | Create a causal chain by selecting steps in order. |
| **L5** | **Evaluate** | **Socratic Open Entry** | Type a reasoning-based answer (min 20 chars). |
| **L6** | **Create** | **Decision Matrix** | Multi-step scenario where choice A affects result B. |

---

## Mechanic Definitions

### 1. Multiple Choice (L1)
*   **UI:** Standard 4-button layout.
*   **Logic:** Immediate feedback. Used for vocabulary, dates, and basic math.

### 2. Multi-Select (L2)
*   **UI:** Checkbox list with a "Confirm" button.
*   **Logic:** Partial credit is NOT given. Must select the exact set of correct properties.

### 3. Tile Matching (L3)
*   **UI:** Two columns of 3 tiles each.
*   **Logic:** Student must tap a tile in Column A then its partner in Column B.

### 4. Sequential Ordering / Connect (L4)
*   **UI:** Draggable or numbered list.
*   **Logic:** Arrange steps of a process (e.g., Water Cycle or a Math Proof).

### 5. Open Reasoning (L5-L6)
*   **UI:** Text Area + "Submit" button.
*   **Logic:** Evaluated by Tier 2/3 AI. Keywords and structural coherence checked.
