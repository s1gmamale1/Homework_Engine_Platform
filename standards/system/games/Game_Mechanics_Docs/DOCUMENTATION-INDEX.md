## Elite Spec Document Structure

Every Elite Spec HTML document follows the same 5-section structure (matching the Territory Conquest reference format):

### Section 1: Core Game Mechanics
- Detailed how-it-works with step-by-step flow
- Grade-based scaling parameters
- Subject-specific implementations
- Session placement rationale

### Section 2: Tier-Based Access Control
- Basic vs Premium feature comparison table
- Tier guarantee statement
- Feature access matrix

### Section 3: Buzan Integration
- Tony Buzan's Radiant Thinking principles applied to the mechanic
- Color hooks, imagery, association chains
- Memory Palace anchor integration (where applicable)

### Section 4: Question Styles & Interaction Mechanics
- PISA level L1-L6 mapping table
- Concept tile type, definition tile type, cognitive target, distractor style
- Stranger Test compliance statement

### Section 5: Victory Conditions & Scoring
- Grade Computation tables (what data feeds the learning curve)
- Session Report Card format (what the student sees)
- Victory tiers (Mastered ★, Proficient ✓, Apprentice △, Novice ○)
- Defeat framing with "Hali emas" (not yet) philosophy
- Special outcome handling (Head-to-Head, Speed Mode, etc.)

## Quick Selection Guide

| When to pick | Pick from |
|---|---|
| Need maximum repetition density (Remember/Understand) | Default: Tile Match, Memory Sprint, Spaced Flashcards |
| Content is narrative-heavy | Default: Story Mode, Why Chain |
| Task requires physical hand-work | Default: Notebook Capture |
| Student in Recovery Session or below PISA L2 | Default (lower cognitive overhead) |
| Need a high-stakes strategic duel | Interactive: Connect Four, Territory Conquest |
| Cross-subject review (Big Boss week) | Interactive: Escape Room, Territory Conquest |
| Student engagement has dipped 3 sessions | Interactive (novelty effect) |
| Topic maps to a spatial/strategic metaphor | Interactive: Puzzle Lock (timelines), Reaction Chain (processes), Territory Conquest (geography) |
| Math-only | Interactive: Blackjack 21 |

## Tier Summary Across All Mechanics

| Feature | Basic Tier | Premium Tier |
|---|---|---|
| **Core learning** | Full 7-phase sessions, all mechanics, full curriculum | Same |
| **Session access** | Phase 3 games only | Phase 3 + on-demand Library |
| **Game modes** | Standard modes only | + Head-to-Head, Speed Mode, Master Mode |
| **Board/grid sizes** | Standard (up to 6 pairs, 3×3 puzzles) | Extended (up to 8 pairs, 5×5 puzzles) |
| **AI feedback** | Tier 1 (rule-based) + Tier 2 (Boss eval) | + Tier 3 (Socratic tutor, detailed rubrics) |
| **Visual polish** | Standard animations | Enhanced particle effects, themes |
| **Difficulty band** | Current PISA level + 1 | Current + 2 (L5-L6 stretch) |
| **Multi-player** | Not available | FFA, One-for-All modes |

## Cost Model by Mechanic

| AI Tier | Mechanics | Cost per student/year |
|---|---|---|
| **Tier 1 ($0)** | Memory Sprint, Spaced Flashcards, Tile Match, Sentence Fill, Memory Palace, Adaptive Quiz, Mystery Box, Tic Tac Toe vs AI, Notebook Capture (Tier 2 basic eval), Connect Four, Memory Match Blitz, Reaction Chain, Puzzle Lock, Territory Conquest | ~$0 |
| **Tier 2 (cents)** | Story Mode, Why Chain, Real-Life Challenge, Escape Room, Notebook Capture (Tier 2 pass/fail) | ~$3-5 |
| **Tier 3 (dollars)** | Final Boss (Tier 3 Socratic on failure), Notebook Capture (Tier 3 detailed rubric, Premium only) | ~$8-15 (Premium only) |

> **Note:** The grading system replaces XP as the primary academic metric. XP remains for metagame rewards (Bilim Tokens, Bilim Bazasi, streaks) but not for tracking learning progress. See `GRADING-SYSTEM.md` for full specification.
