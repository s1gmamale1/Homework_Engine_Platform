Adaptive Quiz

Default Mechanic #7 — IRT-Calibrated Dynamic Difficulty Assessment

Players: 1 Mode: Solo Only Buzan Integrated Bloom's: Apply PISA: 2-3 AI Tier: Tier 1

1. Core Game Mechanics

Adaptive Quiz uses Item Response Theory (IRT) to calibrate questions that adjust difficulty after each answer: correct = harder, wrong = easier. Unlike fixed quizzes, it creates a unique question path per student — finding the exact boundary of ability. This provides ~30% more measurement accuracy than fixed-difficulty assessments.

IRT Parameters

  • Difficulty (theta, θ): -3.0 (easiest) to +3.0 (hardest). θ = 0 = grade-level proficiency.
  • Discrimination (alpha, α): 0.5 to 2.0. Higher α = better at distinguishing students who know vs. don't know.
  • Guessing parameter (c): 0.1 to 0.25 for MC. Free-response = 0.05.

Adaptive Algorithm (5-7 questions per round)

Q1 — Seed (theta = 0): Starts at grade level with high-discrimination question (α ≥ 1.2).

Q2+ — Difficulty Adjustment: Correct → theta += 0.3 (harder). Incorrect → theta -= 0.3 (easier). Capped at ±3.0. System prefers questions with higher α at target theta for maximum information.

Termination: Ends at 7 questions answered, or early convergence (5 questions if standard error < 0.3 for 3 consecutive questions), or ceiling/floor hit (θ = ±3.0 for 2 consecutive questions).

Grade Scaling

  • G1-4: 5 questions. All MC with visuals. Adjustment step = 0.2 (gentler). Min θ = -2.0.
  • G5-7: 5-7 questions. Mix of MC and short-answer. Standard step = 0.3. Full θ range.
  • G8-11: 7 questions. Mix of MC, short-answer, open-response (AI-evaluated). θ > 1.5 may require multi-step reasoning with written justification.

🧠 Buzan Integration: Color-Coded Confidence

  • Color Gradient for Difficulty: θ ≤ -1.0 = calming blue. -1.0 to 0.0 = green (comfortable growth). 0.0 to +1.0 = amber (productive struggle). θ > +1.0 = deep orange-red (maximum stretch). The student sees their proficiency shift with each answer.
  • Radiant Knowledge Map: After each round, a mini mind map shows which concept branches were tested. Correct = solid green nodes. Missed = pulsing amber nodes. Implements Buzan's radiant structure visually.
  • Imagery for Wrong Answers: Incorrect answers display the correct answer alongside a memorable visual metaphor. A missed Newton's third law question shows two ice skaters pushing off each other. Vivid imagery attached to correction creates stronger encoding than text-only.
  • Positive Framing of Difficulty Increase: When θ increases: "Siz kuchliroq savolga tayyorsiz!" — "You're ready for a stronger question!" Difficulty increase is framed as achievement, not punishment.
  • Memory Anchor for Repeated Misses: Missing the same concept type across 3+ rounds triggers a "Memory Anchor" card — one vivid image + one-sentence explanation — appearing as a pre-warmup in the next session.

C. Proficiency Grades

  • Above Grade Level (θ > +0.5): "Zo'r! Siz sinf darajasidan yuqorisiz!" — Theta displayed on rising arrow. PISA tier upgraded if consistent across 3+ rounds. Grade: Proficient ✓ or Mastered ★.
  • At Grade Level (-0.5 ≤ θ ≤ +0.5): "Yaxshi! Siz kutayotgan darajadasiz." — Theta at grade-level line. Student confirmed at current PISA tier.
  • Below Grade Level (θ < -0.5): "Hali emas — lekin siz o'sib boryapsiz." — Theta with upward arrow showing growth potential. No PISA tier downgrade from a single round (requires 3 consecutive below threshold).

D. No Failure State

Adaptive Quiz cannot be failed — every round produces valid measurement data. Even θ = -3.0 is useful information (tells the system the student needs foundational support). The "Hali emas" framing applies only when θ drops below the student's current PISA tier threshold, signaling targeted remediation is needed.

E. Long-Term Tracking

  • Theta trajectory: Plotted across all rounds. Upward trend = positive velocity. Flat = plateau. Downward = investigation needed.
  • PISA level consistency: Student must demonstrate same PISA level across 3 consecutive rounds before the level is "confirmed."
  • Subject heatmaps (Premium): Per-topic theta estimates within a subject — "Algebra: θ=+0.8, Geometry: θ=-0.3, Statistics: θ=+0.1" — identifies specific strength/weakness branches.
  • PISA projection (Premium): Predictive model estimates when the student will reach the next PISA level based on their theta trajectory over the past 10 rounds.
