# Phase 3: Game Breaks

Endi haqiqiy amaliyotni boshlaymiz! Preview qismida o'rgangan qoidalaringizni o'yinlar orqali mustahkamlab olasiz. Har bir vazifani diqqat bilan bajaring.

## Game 1: Adaptive Quiz
Ushbu bosqichda siz modul qatnashgan tenglama va tengsizliklarni mustaqil ravishda yechishingiz kerak. Barcha javoblarni o'zingiz yozing (variantlar berilmaydi). Har bir misolning yechilish jarayonini daftaringizga yozib, rasmga oling (Notebook Capture).

**Item 1 (Easy):** `|x| = 12` tenglamasining musbat ildizini yozing.
**Javob:** 12
`[Bloom: L2 | PISA: L1]`

**Item 2 (Easy):** `|y| = -8` tenglamasi nechta yechimga ega? (Faqat raqam bilan yozing)
**Javob:** 0
`[Bloom: L2 | PISA: L1]`

**Item 3 (Medium):** `|x - 5| = 2` tenglamasining eng kichik ildizini toping.
**Javob:** 3
*(Tushuntirish: x - 5 = 2 -> x = 7; x - 5 = -2 -> x = 3)*
`[Bloom: L3 | PISA: L2]`

**Item 4 (Medium):** `|2x + 1| = 7` tenglamasining ildizlari yig'indisini toping.
**Javob:** -1
*(Tushuntirish: 2x + 1 = 7 -> x = 3; 2x + 1 = -7 -> x = -4. Yig'indi: 3 + (-4) = -1)*
`[Bloom: L3 | PISA: L2]`

**Item 5 (Hard):** `|x| < 4` tengsizlikni qanoatlantiruvchi eng katta butun sonni toping.
**Javob:** 3
*(Tushuntirish: -4 < x < 4. Bu oraliqdagi eng katta butun son 3)*
`[Bloom: L4 | PISA: L2]`

**Item 6 (Hard):** `|3x - 2| ≤ 10` tengsizlikni qanoatlantiruvchi butun sonlar nechta?
**Javob:** 7
*(Tushuntirish: -10 ≤ 3x - 2 ≤ 10 -> -8 ≤ 3x ≤ 12 -> -2.66 ≤ x ≤ 4. Butun sonlar: -2, -1, 0, 1, 2, 3, 4. Jami 7 ta)*
`[Bloom: L4 | PISA: L3]`

## Game 2: Tile Match
Chap tomondagi modulli ifoda yoki tenglamani o'ng tomondagi unga teng kuchli ifoda yoki yechim bilan moslashtiring.

| Chap tomon (Ifoda/Tenglama) | O'ng tomon (Yechim/Ma'no) |
|-----------------------------|---------------------------|
| `|x| = 5` | `x = 5` yoki `x = -5` `[Bloom: L1 | PISA: L1]` |
| `|x| = -3` | Yechimga ega emas `[Bloom: L2 | PISA: L1]` |
| `|x| ≤ 2` | `-2 ≤ x ≤ 2` `[Bloom: L2 | PISA: L2]` |
| `|x - 1| = 0` | `x = 1` `[Bloom: L2 | PISA: L1]` |
| `|-7|` | `7` `[Bloom: L1 | PISA: L1]` |
| `|x| < 0` | Yechimga ega emas `[Bloom: L2 | PISA: L2]` |

## Game 3: Sentence Fill
Quyidagi matematik qoida va yechish bosqichlaridagi tushirib qoldirilgan qismni to'ldiring.

**Item 1:** `|x - 2| = 5` tenglamani yechish uchun ikkita holatni qaraymiz: `x - 2 = 5` va `x - 2 = ___`
**Javob:** -5
`[Bloom: L2 | PISA: L1]`

**Item 2:** `|2x| ≤ 6` modulli tengsizlik quyidagi qo'sh tengsizlikka teng kuchli: `___ ≤ 2x ≤ 6`
**Javob:** -6
`[Bloom: L3 | PISA: L2]`

**Item 3:** Modul qoidasiga ko'ra, agar `a < 0` bo'lsa, `|a| = ___` bo'ladi.
**Javob:** -a
`[Bloom: L1 | PISA: L1]`

**Item 4:** Modul masofani bildirgani uchun `|x + 4| = -2` tenglamasining javobi `___` bo'ladi.
**Javob:** Yechim yo'q (yoki 0 ta)
`[Bloom: L2 | PISA: L2]`

**Item 5:** `|x| < 7` tengsizligi oraliq ko'rinishida `-7 < x < ___` deb yoziladi.
**Javob:** 7
`[Bloom: L2 | PISA: L1]`
