**Front:** Sonning moduli
**Back:** |a| = a (agar a ≥ 0), |a| = -a (agar a < 0). Misol: |-5| = -(-5) = 5.

**Front:** Musbat sonning moduli
**Back:** Musbat sonning moduli shu sonning o'ziga teng. Misol: |3| = 3.

**Front:** Manfiy sonning moduli
**Back:** Manfiy sonning moduli unga qarama-qarshi songa teng. Misol: |-2| = -(-2) = 2.

**Front:** Nolning moduli
**Back:** Nolning moduli nolga teng. Misol: |0| = 0.

**Front:** Modulning geometrik ma'nosi
**Back:** |a| bu son o'qida 0 nuqtadan a nuqtagacha bo'lgan masofadir. Misol: |-2| = 2 qadam masofa.

**Front:** |x| = a tenglama (a > 0)
**Back:** Ikkita ildizga ega: x1 = a, x2 = -a. Misol: |x| = 7 → x = 7, x = -7.

**Front:** |x| = a tenglama (a < 0)
**Back:** Haqiqiy yechimga ega emas, chunki modul manfiy bo'lmaydi. Misol: |x| = -5 → Yechim yo'q.

**Front:** |x| ≤ a tengsizlik (a > 0)
**Back:** -a ≤ x ≤ a qo'sh tengsizlikni bildiradi. Misol: |x| ≤ 2.5 → -2.5 ≤ x ≤ 2.5.

**Front:** |x| < a tengsizlik (a > 0)
**Back:** -a < x < a qo'sh tengsizlikni bildiradi. Misol: |x| < 3 → -3 < x < 3.

**Front:** |x| ≥ a tengsizlik (a > 0)
**Back:** x ≥ a yoki x ≤ -a nurlarning birlashmasini bildiradi. Misol: |x - 1| ≥ 2 → x - 1 ≥ 2 yoki x - 1 ≤ -2.
