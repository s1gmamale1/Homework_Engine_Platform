# Story Mode — Narrative Concept Delivery with Comprehension Gates

**Default Mechanic #6**
**Bloom's Level:** Understand
**PISA Level:** 2-3
**AI Tier:** Tier 2 (cents/student/year)
**Research Basis:** Narrative Retention (Willingham, 2004), Expected Gain: 22x better retention
**Session Phase:** Phase 2 (Concept Delivery)

---

## 1. Core Game Mechanics

Story Mode delivers textbook content as a **narrative with a 4-beat dramatic arc: Problem → Struggle → Discovery → Solution**. Research shows stories are up to 22x more memorable than isolated facts.

### 4-Beat Arc Structure (5-7 minutes total)

**Beat 1 — Problem (60-90s):** Introduces an Uzbek character facing a challenge requiring the target concept to resolve.

**Gate 1 — Comprehension Check (30s):** 1-2 IELTS-style questions about the Problem beat. Must answer ≥1 correctly.

**Beat 2 — Struggle (90-120s):** Character tries everyday reasoning and fails, creating cognitive tension.

**Gate 2 — Comprehension + Prediction (30s):** Questions about the Struggle beat + open prediction logged for analysis.

**Beat 3 — Discovery (90-120s):** Mentor figure reveals the target concept as the key to solving the problem.

**Gate 3 — Concept Comprehension (30-45s):** Tests concept understanding. Must pass ≥1 of 2 questions.

**Beat 4 — Solution (30-60s):** Character applies the concept. Concept explicitly named and summarized.

### The "Stranger Test"
Every story must pass: a test player with zero subject knowledge must follow and identify the correct solution using only narrative context clues.

### Grade Scaling
- **G1-4:** 3-beat arc. Shorter passages (40-60s). Single MC with visuals. Read aloud with highlighting.
- **G5-7:** Full 4-beat arc. 2 questions per gate (literal + inference).
- **G8-11:** Full 4-beat + 5th beat: **Extension** — harder problem in new context. Short-answer AI-evaluated.

### Cultural Anchoring Mandate
Uzbek names, settings (Bukhara courtyards, Samarkand schools, Fergana farms), daily life references (plov, Navruz, bazaars). Validation gate before shipping.

## 2. Tier-Based Access Control

| Feature | Basic Tier | Premium Tier |
|---------|-----------|--------------|
| **Session Access** | Phase 2 concept delivery | Same + on-demand Library access |
| **Story Format** | Text + static illustrations | + Animated transitions, ambient sound, voice acting |
| **Gate Questions** | 2 per gate (6 total) | 3 per gate (9 total) + Extension gate for G8-11 |
| **Story Branching** | Linear — one path | Branching — choices at Gate 2 change Discovery approach |
| **Character Customization** | Fixed character | Choose name, appearance, home city |
| **Extended Stories** | Single-concept (5-7 min) | "Chapter" mode — multi-concept saga, 3-4 sessions |
| **Story Creator** | Not available | G9-11 co-create stories with AI |

> **Basic Tier Guarantee:** All students receive the complete 4-beat narrative with comprehension gates. Premium adds cinematic polish, branching paths, and extended sagas — enrichment that deepens engagement but is not required for concept mastery.

## 3. Buzan Integration: Narrative Radiance

- **Color as Emotional Anchor:** Problem = warm amber, Struggle = blue-gray, Discovery = green/gold, Solution = warm white/green. The color progression creates an emotional arc for memory encoding.
- **Vivid Imagery (Picture Power):** At least 2 "sticky images" per story. "Tiny green factories" (not just "organelles").
- **Association Chains (Radiant Links):** Story characters become "radiant hubs." Subsequent stories with same character link back to prior knowledge.
- **Personal Relevance (Self Principle):** "What would YOU do?" prompts trigger self-reference effect — 2-3x better recall.
- **Sensory Storytelling:** At least one non-visual sensory detail per beat. Multi-sensory encoding creates redundant retrieval paths.

## 4. Question Styles & Interaction Mechanics

| PISA Level | Gate Question Style | Cognitive Target | Pass Threshold | Feedback on Failure |
|------------|-------------------|-----------------|----------------|--------------------|
| **L1** | Literal MC, 3 options, visuals | Direct text retrieval | 1 of 1 | Passage reread with target highlighted |
| **L2** | Inference MC, connect sentences | Read between the lines | 1 of 2 | Target sentences extracted |
| **L3** | Application MC with explanation | Apply to modified scenario | 1 of 2 | Concept re-explained with new example |
| **L4** | Analysis: select from process steps | Identify causal chain | 1 of 2 | Process diagram highlighted |
| **L5** | Evaluation short-answer, AI-evaluated | Evaluate competing claims | ≥3/5 on rubric | Model answer after 2 failures |
| **L6** | Creation open-ended, multi-paragraph | Synthesize with novel constraints | ≥4/5 on rubric | Peer exemplar shown |

> **Gate Design Philosophy:** Gates are comprehension checkpoints, not exams. A student who fails needs the story revisited. Even after 2 failures, the student can proceed (concept flagged for later review).

## 5. Victory Conditions & Scoring

### A. Grade Computation

This mechanic feeds **Type C (Outcome-Based)** data:

| What Gets Recorded | Source | Curve Impact |
|---|---|---|
| Gate accuracy (%) | Correct gate answers / total questions | Primary — determines Level |
| Reading pace | Seconds per beat completed | Secondary — feeds Efficiency metric |
| Retries needed | Gate re-reads triggered | Tertiary — fewer retries = higher confidence |
| Attempts (cumulative) | Total story sessions | Feeds curve-fitting (y = K × attempts^n) |

### B. Session Report Card

```

Read pace: 45s/beat (getting faster ✓)
~2 sessions to Mastered
```

### C. Outcome Grades

- **Master Reader (all gates first attempt):** Gold bookmark. Grade: Proficient ✓
- **Strong Reader (1 retry):** Silver bookmark
- **Completed (2+ retries):** Bronze bookmark. "Hali emas — lekin siz yetib keldingiz!"
- **Incomplete (gate failed after 2 retries):** Story pauses, Solution shown in summary. "Bu hikoya sizni kutib turadi." No grade penalty.

### D. "Hali Emas" (Not Yet) Framing

A student who cannot pass a gate has identified a comprehension gap — not failed. The story pauses, reveals the ending, and promises to return. No grade penalty. Concept flagged for review in a different mechanic (Adaptive Quiz or Final Boss).
